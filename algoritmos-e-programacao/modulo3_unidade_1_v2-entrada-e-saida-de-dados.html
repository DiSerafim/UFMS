print(), input() e eval()

print()

É usada para imprimir aquilo que é enviado no argumento (entre parênteses).

Ex.:
>>> print(0)
0
>>> x = 2
>>> print(x)
2
>>> print([1, 2, 3])
[1, 2, 3]
>>> print('Olá')
Olá

É possível também passar mais de um argumento para a função.
Ex.:
>>> h = 13
>>> print('Agora são', h, 'horas')
Agora são 13 horas

Outros argumentos da função print();
>>> dia = 1
>>> mes = 12
>>> ano = 1990
>>> print(dia, mes, ano, sep='/')
1/12/1990
>>> print(dia, mes, ano, sep='/', end='.\n')
1/12/1990.

>>> x = 2
>>> y = 3
>>> print("x = ", x, "y = ", y)
x = 2 y = 3
>>> print('x = ', x, 'y = ', y, sep='|')
x = |2|y = |3

input()

É usado para solicitar dados do usuário no shell.
A função input sempre retorna uma string, independente do valor digitado pelo usuário.

Ex.:
>>> nome = input('Digite seu nome')
Digite seu nome Diego Serafim
>>> nome
' Diego Serafim'

>>> x = input('Digite o valor de x: ')
Digite o valor de x: 6
>>> x
'6'
>>> type(x)
class 'str'

eval()

É possível explicitamente instruir a linguagem para avaliar uma string como uma expressão Python.

Ex.:
>>> eval('3')
3
>>> eval('3 + 4')
7
>>> eval('len([1, 2, 3])')
3
>>> eval('3 > 4')
False
>>> x = input('Valor: ')
Valor: 6
>>> eval(x)
6
>>> eval('x')
'6'

Exercício

Fazer um programa em Python em que o usuário informa uma temperatura em Celsius, e o programa emite a temperatura em Fahrenheit.
Usar a fórmula: F = 1.8 * C + 32.

c = input('Digite a temperatura em Celsius: ')
c_temp = eval(c)
f = 1.8 * c_temp + 32
print(f, 'Graus Fahrenheit.')
