Programas em Python

Um programa em Python é um conjunto de instruções que são executadas em ordem, seguindo os conceitos sobre algoritmos que vimos antes:
- sequenciamento.
- estruturas de condição.
- estruturas de repetição.

Em vez de usarmos o shell interativo para executar cada instrução em Python, iremos implementar as instruções de um programa em um ou mais arquivos com extensão .py.

No meu caso, irei usar o terminal do meu Linux:
criarei o programa 'nome.py' e executarei através do terminal.

└─$ python3 0_3_ex01.py
