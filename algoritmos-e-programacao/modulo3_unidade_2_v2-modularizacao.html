Funções - Introdução

Uma função é um agrupamento de instruções que podem ser executadas mais de uma vez em um programa.
É possível passar a uma função parâmetros de entrada, que podem ser diferentes a cada vez que a função é executada.

Finalidade das funções

Criamos funções por dois motivos:
- Reutilização de código. - Modularização.

Tipos de funções

Funções internas (builtin). Já definidas pelo Python
Funções definidas pelo usuário.

Criando funções em Python

Definimos uma nova função usando a instrução def, de acordo com a sintaxe a seguir:
def <nome_função>(argumento):
<instruções>

Ex.: função para exibir uma mensagem:
def mensagem():
print("Bóson Treinamentos em Tecnologia!")

A instrução def cria um objeto e atribui um nome a ele.

Criando função em Python com argumentos

Definimos uma nova função usando a instrução def, de acordo com a sintaxe a seguir:
def <nome_função>(argumento):
<instruções>
Ex.: função para somar dois números:
def soma(a, b):
return a + b
---
>>> def soma(a, b):
... return a + b
>>> soma(9, 9)
18

Ex.1:
valores = [1, 2, 3, 4, 5]
def quadrado(valores):
quadrados = []
for x in valores:
quadrados.append(x ** 2)
return quadrados

resultados = quadrado(valores)
for y in resultados:
print(y)
