Listas, tuplas e operadores

Listas
Uma lista é uma sequência de objetos.
Objetos podem ser de diferentes tipos:
- números.
- string.
- outras listas. Etc.
Em python, uma lista é representada por objetos separados por vírgula, envoltos por colchetes [].
Exemplos:
>>> pets = ['cão', 'gato', 'peixe']
>>> pets
['cão', 'gato', 'peixe']
>>> l = [1, 'abc', [], [1, 2]]
>>> l
[1, 'abc', [], [1, 2]]

Listas são Mutáveis Uma diferença importante entre listas e strings é que listas são mutáveis, isto é, seu conteúdo pode variar.
Exemplo:
>>> pets
['cão', 'gato', 'peixe']
>>> pets[1] = 'galinha'
>>> pets
['cão', 'galinha', 'peixe']

Tuplas são Imutáveis
Tuplas são iguais listas, mas imutáveis.
Usamos parênteses (()) ao invés de colchetes ([]) para criá-las.
Exemplo:
>>> dias = ('seg', 'ter', 'qua')
>>> dias
('seg', 'ter', 'qua')
>>> dias[2] = 'qui'
TypeError: 'tuple' object does not support item assignment

Listas em Python possuem métodos que podem ser chamados a partir de uma lista já criada:
>>> pets
['cão', 'galinha', 'peixe']
>>> pets.append('gato')
>>> pets
['cão', 'galinha', 'peixe', 'gato']

Outros métodos:
count(), confere os elementos
index(), retorna o elemento conforme a posição passada
insert(), inserir elementos
pop(),
remove(),
reverse(),
sort()

Exemplos:
my_list = [2, 4, 1, 3, 4, 5]

# Exemplo de uso dos métodos
my_list.count(4) # Conta quantas vezes o número 4 aparece
my_list.index(3) # Encontra o índice do número 3
my_list.insert(0, 6) # Insere o número 6 no início da lista
my_list.pop(2) # Remove o elemento no índice 2
my_list.remove(4) # Remove a primeira ocorrência do número 4
my_list.reverse() # Inverte a ordem dos elementos
my_list.sort() # Classifica a lista em ordem crescente
