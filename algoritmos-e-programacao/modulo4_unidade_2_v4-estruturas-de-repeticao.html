Listas Multidimensionais

No caso de duas dimensões, podemos criar uma lista de listas: 

            m = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
Assim,
m[0] corresponde à primeira linha (lista [1, 2, 3]),
m[1] à segunda linha (lista [4, 5, 6]) e
m[2] à terceira linha (lista [7, 8, 9]).

Usar um segundo índice permite acessar uma coluna: 

            >>> m = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
            >>> m[0][0]
            1
            >>> m[1][2]
            6

Exemplo:

Implementar uma função em Python que verifica se uma matriz bidimensional é simétrica ou não. 

        >>> m = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
        >>> def simetrica(m):
        ...     nlinhas = len(m)
        ...     ncolunas = len(m[0])
        ...     for i in range(nlinhas):
        ...         for j in range(i+1, ncolunas):
        ...             if m[i][j] != m[j][i]:
        ...                 return False
        ...     return True
        ... 
        >>> simetrica(m)
        False
        >>> simetrica([[1, 2, 3], [2, 4, 5], [3, 5, 3]])
        True
