Strings

Além de variáveis do tipo numérico (int e float), e do tipo lógico (bool), Python permite definir variáveis de tipos mais complexos.

O tipo 'str' é usado para representar e manipular texto ou uma sequência de caracteres, incluindo espaço em branco, pontuação e símbolos diferentes.

Uma string é criada como uma sequência de caracteres envolvida por aspas (simples ou dupla):
>>> a = 'Algoritmos e '
>>> A = "Programação"
>>> a, A
('Algoritmos e ', 'Programação')

É possível usar alguns operadores já vistos com strings.
>>> s == 'abc'
True
>>> t = 'def'
>>> s < t
True
>>> s + t
'abcdef'
>>> s * t
Traceback (most recent call last):
File "", line 1, in
TypeError: can't multiply sequence by non-int of type 'str'
>>> s * 2
'abcabc'
>>> ch = 'b'
>>> ch in s
True
>>> ch is str('b')
True

operadores comuns que não funcionam com strings:
-, /, //, %, **

Operadores importantes
len() → retorna o tamanho da string.
[] → operador de indexação.

Exemplos:
>>> s = 'abcd'
>>> len(s)
4
>>> s[0]
'a'
>>> s[3]
'd'
>>> s[len(s)]
Traceback (most recent call last):
File "", line 1, in
IndexError: string index out of range

Indexação
Apesar do índice variar de 0 até o tamanho da string-1, é possível usar índices negativos.
Índices negativos acessam os caracteres do lado direito da string.
Exemplos:
>>> s = 'abcd'
>>> s[-1]
'd'
>>> s[-2]
'c'
>>> s[len(s[-1])]
'b'

É possível usar também substrings por meio da indexação:
>>> s = 'abcd'
>>> s[0:2]
'ab'
>>> s[-4:2]
'ab'
>>> s[:3]
'abc'
>>> s[-1]
'd'

Métodos >>> s.find(a)
-1
>>> s.count(a)
0
>>> s.upper()
'ABCD
>>> s = s.lower()
>>> s
'abcd'
>>> s.capitalize()
'Abcd'
