Matrizes

É uma coleção bidimensional de elementos, onde cada elemento é identificado por dois índices: um índice de linha e um índice de coluna. Matrizes são uma forma comum de armazenar e manipular dados tabulares ou tabelas em Python.
>>> matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
>>> matriz
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]

Podemos usar uma lista de lista para apresentar uma matriz 2D: - Cada elemento da lista contém uma linha da matriz;
>>> matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
- Cada linha corresponde a uma lista com os elementos das colunas da matriz.
>>> matriz[0] → [1, 2, 3]
>>> matriz[1] → [4, 5, 6]
>>> matriz[2] → [7, 8, 9]
-------
O acesso aos valores matriz é feito da seguinte forma: [Linha] [Coluna]
>>> matriz[0][0] → 1
>>> matriz[1][0] → 4
>>> matriz[2][1] → 8
-------
>>> type(matriz)
class 'list'
