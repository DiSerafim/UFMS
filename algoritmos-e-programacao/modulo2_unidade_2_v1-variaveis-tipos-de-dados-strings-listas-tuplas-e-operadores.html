Variáveis

É um nome que é atribuído a um objeto.
Ex.:
>>> x = 3
>>> x
3
3 → objeto número
x → variável
Obs.: Operador de atribuição: =

Formato geral:

<'variável'> = <'expressão'>
>>> x = 3 + 3
>>> 2 * x
12
>>> y = x - 2
>>> y
4
>>> x = y
>>> x
4

Os nomes de variáveis podem conter caracteres em minúsculos(a-z), maiúsculo(A-Z), underscore(_), e exceto para o primeiro caractere, os dígitos de 0 a 9.

Ex. Variáveis válidas:
nome
_salario
nota1
Variavel2
primeiroNome
enderecoIP
resultado_final


Ex. Variáveis inválidas:
2variavel →(Começa com um dígito)
meu-salario →(Usa um hífen em vez de sublinhado)
for →(Usa uma palavra reservada do Python)
@quantidade →(Usa um caractere especial @)
espaço em branco →(Nomes de variáveis não podem conter espaços)
class →(Usa uma palavra reservada do Python)
