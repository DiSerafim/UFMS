Expressões aritméticas, lógicas e seus operadores

Operações cujos operadores são aritméticos e cujos operandos são números, constantes ou variáveis do tipo numérico.

Operadores aritméticos:
+ Adição
- Subtração
* Multiplicação
/ Divisão
// Divisão Inteira
% Resto da Divisão
** Potência

Expressões Aritméticas

Exemplos:
>>> 2**3
8
>>> 5//2
2
>>> 5/2
2.5
>>> 4 / (2 + 3)
0.8

Precedência de Operador

Mesma regra da álgebra:
Multiplicação e divisão têm precedência sobre soma e subtração.
>>> 3 + 2 * 2
7
Parênteses são usados para modificar a precedência
>>> (3 + 2) * 2
10
Por último, avaliação da esquerda para a direita
>>> 2 - 3 + 1
0

Tipos de Dados

int → números inteiros (2, 3, 123, 0)
float → números com casas decimais (0,5, 1,2)
Soma / subtração / multiplicação de dois inteiros → resultado é um int (2, 3, 123, 0)
Se ao menos um float aparece na expressão → resultado é um float (0,5, 1,2)
Divisão de dois inteiros → resultado é um float (0,5, 1,2)

Quociente e Resto

Operadores // e %
>>> 5 // 2
2
>>> 5 % 2
1
Tanto // quanto % geram como resultado um int

Funções Matemáticas

abs() → retorna valor absoluto de um número:
>>> abs(-2)
2

min() → retorna o mínimo de um conjunto de valores:
>>> min(3,432)
3

max() → retorna o máximo de um conjunto de valores:
>>> max(3,432)
432
