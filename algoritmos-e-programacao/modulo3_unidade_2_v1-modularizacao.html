Definição de funções

Python permite que o programador crie suas próprias funções.
É útil quando precisamos executar uma sequência de instruções várias vezes, em diferentes partes do programa.
Permite modularizar o programa, facilitando o reúso e manutenção.

Primeira função
Suponha que queremos definir uma função que calcula f(x):
f(x) = x² + 1
Podemos definir a função como segue:
def f(x):
res = x**2 + 1
return res

>>> def f(x):
...res = x**2 + 1
...return res
>>> f(4)
17

Formato

O formato geral para definição de funções em python é:
def <nome da função>(<parâmetros>):
<instruções com indentação >
...
return <valor> (opcional)

Função com vários parâmetros

Função que calcula o preço de um produto atualizado pela taxa de juros:

def juros(preco, juros):
res = preco * (1 + (juros / 100))
return res

>>> def juros(preco, juros):
... res = preco * (1 + (juros / 100))
... return res
>>> juros(650, 60)
1040.0

print() e return()
Qual a diferença entre as duas funções a baixo?

Ex1.:
def f(x):
res = x ** 2 + 1
return res

>>> def f(x):
... res = x ** 2 + 1
... return res
>>> f(81)
6562

return é uma instrução usada dentro de funções para retornar um valor a quem chamou a função.
É usado para passar um valor de volta ao ponto de chamada da função, permitindo que esse valor seja armazenado em uma variável ou usado em expressões.
O valor retornado por return é frequentemente usado para resultados de cálculos ou para tomar decisões com base em valores calculados dentro de uma função.

Ex2.:
def f(x):
res = x ** 2 + 1
print(res)

>>> def f(x):
... res = x ** 2 + 1
... print(res)
>>> f(81)
6562

print() é uma função em Python usada para exibir informações na saída padrão, que geralmente é a tela do computador.
É usado para mostrar valores, mensagens ou resultados de cálculos na tela para que o usuário possa vê-los.
print() não retorna um valor para ser usado em cálculos ou operações posteriores. Sua função é puramente de saída.

Ordem de definição

Ao definir uma função em Python, é importante sempre defini-la antes de executar, na ordem sequencial de cima para baixo no arquivo.

Ex. erro:
print(j(3))
def j(x):
return x ** 2 + 1

print(j(3))
Traceback (most recent call last):
File "", line 1, in
NameError: name 'j' is not defined
>>> def j(x):
File "", line 1
def j(x):
IndentationError: unexpected indent

Ex. Sem erro com função definidas sem executá-las:
def l(x):
return m(x)

def n(x):
return x ** 2 + 1

>>> def l(x):
... return m(x)
...
>>> def n(x):
... return x ** 2 + 1
...
