Estruturas de repetição - while

O comando while tem uma estrutura parecida com o teste condicional de uma via: 

            if <condição>:
                <bloco de instruções indentado>
            <bloco de instruções não indentado>


            while <condição>:
                <bloco de instruções indentado>
            <bloco de instruções não indentado>

Fluxograma:


        [INÍCIO] --> [Iniciar While]
                    ↓
                [Condição Verdadeira?]
                    |
                    ↓ (Não)
                [Fim do Loop] --> [FIM]

        |
        ↓ (Sim)
        
        [Executar Código]
        ↓
        [Verificar Condição]

Explicação:

1. O fluxograma começa no "Início".
2. Ele entra no loop while.
3. Dentro do loop, a condição é verificada. Se a condição for verdadeira, ele executará o código dentro do loop.
4. Após a execução do código, a condição é verificada novamente.
5. Se a condição for falsa, o loop while é encerrado e o fluxo segue para o "Fim do Loop" e, eventualmente, para o "Fim".
6. Se a condição for verdadeira, o loop continuará a executar o código e a verificar a condição até que a condição se torne falsa, momento em que o loop será encerrado.

O comando while é útil quando não sabemos quantas vezes um bloco deverá ser repetido.
Exemplo:
Encontre o maior valor de n tal que n! < L.
Se L = 20, então o maior valor de n é 3,
já que 3! = 6 e 4! = 24. 

            >>> def nfat(L):
            ...     n = 0
            ...     fat = 1
            ...     while fat <= L:
            ...             n += 1
            ...             fat *= n
            ...     return n-1

Exemplo:

1. Pedir para o usuário digitar um número inteiro positivo. Caso contrário, emitir uma mensagem e pedir novamente. 

        num = eval(input('Digite um número positivo: '))
        while num < 0:
            num = eval(input('Digite um número positivo: '))
        print('Valeu *^*')

2. Armazenar nomes de pessoas em uma lista enquanto o usuário digitar strings diferentes de vazio. 

        l = []
        nome = input('Digite um nome: ')
        while nome != '':
            l.append(nome)
            nome = input('Digite um nome: ')

        for i in l:
            print(i)

Loop Infinito

É aquele que é executado "Para sempre!": 

            while True:
                <bloco de instruções indentado>
São úteis quando o programa deve fornecer um serviço indefinitivamente.
Para sair do loop, deve-se digitar Ctrl+C no teclado.
