Estruturas de repetição - for

Estruturas de controle de fluxo que permite repetir uma sequência de comandos.
Número de repetições pode ser indeterminado, porém deve ser finito.

Comando FOR

for <variável> in <sequência>:
<bloco de código indentado>
<bloco de código não indentado>

A variável <sequência> deve ser um objeto do tipo string, lis, range ou qualquer outro tipo que possa ser iterado.
Valores sucessivos de <sequência> são atribuídos a <variável>, executando <blocos de código indentado> para cada valor de <variável>.

Exemplo: 

            l = ['cão', 'gato', 'coelho']
            for i in l:
                print(i)
        
            s = 'algoritmos'
            for c in s:
                if c in 'aeiou':
                    print(c)

Função range(). Formato:
range(start, stop, step) 

            for x in range(10):
                print(x)
        
            for y in range(1, 20, 2):
                print(y)
        
            l = ['a', 'b', 'c']
            for i in range(len(l)):
                print(l[i])

Acumuladores

Exemplo:
Como calcular a soma de todos os números pares de 1 a 100? 

            acum = 0
            for x in range(1, 101):
                if x % 2 == 0:
                    acum = acum + x
            print(acum)

Loops Aninhados

Exemplo:
Dada uma lista de strings como segue:
str_list = ['João', 'Roberto', 'Rafael']
Retornar todas as vogais que ocorrem nos elementos acima.
Resp.: o, o, o, e, o, a, a, e 

            str_list = ['João', 'Roberto', 'Rafael']

            for nome in str_list:
                for c in nome:
                    if c in 'aeiou':
                        print(c)
