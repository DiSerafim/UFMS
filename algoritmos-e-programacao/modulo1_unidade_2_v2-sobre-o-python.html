Rodar os programas em Python

* Interativamente.
* Por meio de arquivos de código-fonte.

Rodar Interativamente

Prompt interativo.
Digite python3 em um terminal para entrar no Prompt.
└─$ python3
Python 3.11.5 (main, Aug 29 2023, 15:31:31) [GCC 13.2.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>

Após entrar no modo interativo, digite os comandos desejados, ex.:
>>> print("DiSerafim")
DiSerafim
Para sair do modo interativo digite 'exit()' ou pressione Ctrl + D

Usar arquivos

- Use arquivos para armazenar seu código-fonte e salvá-lo para uso posterior.
- Use editor de textos de sua preferência.
- IDLE é um IDE muito utilizado.
- Você pode rodar o código a partir da linha de comandos ou a partir da interface do IDLE.
- Na linha de comandos do sistema operacional digite 'python3 arquivo.py'
Ex.:
└─$ python3 modulo1_unidade_2_v2-sobre-o-python.py
Arquivo funcionando!

Outros IDEs

- Eclipse, com plugin-in PyDev.
- Netbeans for Python.
- PythonWin.
