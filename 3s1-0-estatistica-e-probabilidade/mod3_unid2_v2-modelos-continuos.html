Exercício 1.

Bernoulli Binomial.

Consumidores de chocolate:

  • 60% preferem marca A.
  • 40% preferem marca B.
  • 6 consumidores escolhidos ao acaso.
  • a) Probabilidade de exatamente 3 preferirem A?
  • b) Probabilidade de pelo menos 3 preferirem A?
  • c) Probabilidade de no máximo 1 preferir A?
  • d) Média de desvio padrão?

Binomial

  • P(x = k) = (n/k)·pᵏ·qⁿ⁻ᵏ
  • N = 6
  • p = 0,6
  • q = 0,4
  • a) k = 3
    • P(k = 3) = (6/3)·0,6³·0,4³ = 0,2765 = 27,65%„
  • b) k ≥ 3
    • P(k≥3) = P(k=3) + P(k=4) + P(k=5) + P(k=6)
    • P(k=3) + (6/3)·0,6³·0,4³ = 0,2765 = 27,65%„
    • P(k=4) + (6/4)·0,6⁴·0,4² = 0,3110 = 31,10%„
    • P(k=5) + (6/5)·0,6⁵·0,4¹ = 0,1866 = 18,66%„
    • P(k=6) + (6/6)·0,6⁶·0,4° = 0,0467 = 4,67%„
    • P(k≥3) = 0,8208
  • c) k ≤ 1
    • P(k ≤ 1) = P(k=0) + P(k=1)
    • P(k = 0) = (6/0)·0,6°·0,4⁶ = 0,0040 = 0,40%
    • P(k = 1) = (6/6)·0,6¹·0,4⁵ = 0,0369 = 3,69%
    • P(k ≤ 1) = 0,04096 = 4,1%
  • d) μ(x) = n·p
    • σ²(x) = n·p·(1-p)
    • μ = 6·0,6 = 3,6
    • σ² = 6·0,6·0,4 = 1,44
Exercício 2.

Poisson.

Livro de Estatística:

  • 400 páginas.
  • 25 erros.

Qual a probabilidade de:

  • a) Uma página conter exatamente um erro?
  • b) A soma das erros em duas páginas ser 2?
  • c) Na primeira página não haver erro e na última página haver exatamente um erro?
  • Calcule o número médio de erros por página.
  • Os eventos são independentes.
  • λ é constante.
  • Distribuição de Poisson.

Média:

  • λ = 25/400(erros/páginas) = 1/16
  • Probabilidade:
  • P(X = k) = (e⁻λᵗ·(λt)ᵏ)/k!
  • ...
  • a) P(x=1)
    • P(X = 1) = (e⁻⁽¹⁄¹⁶⁾·¹((1/16)·1)¹)/1! ≃ 0,0587 = 5,87%
  • b) P(x=2)
    • P(X = 2) = (e⁻⁽¹⁄¹⁶⁾·²((1/16)·2)²)/2! ≃ 0,00689 = 0,69%
  • c) P(1ª não E 400ª sim)?
    • P(1ªñ ∩ 400ªs) = P(1ªñ)·P(400ªs)
    • P(1ªñ ∩ 400ªs) = ((e⁻⁽¹⁄¹⁶⁾·1·((1/16)·1)°)/0!) · (e⁻⁽¹⁄¹⁶⁾·¹·((1/16)·1)¹)/1!
    • P(1ªñ ∩ 400ªs) ≃ 0,0552 = 5,52%
Exercício 3.

Guindaste.

Resistência ao peso →

  • μₓ = 300kg
  • σₓ = 10kg
  • Quatro caixas.
  • μᵤ = 70kg
  • σᵤ = 13kg

Qual a probabilidade de acionar o sistema de emergência?

  • Soma de variáveis.
  • μ(w) = μ(y+y+y+y) =
  • 4μ(y) = 4·70 = 280kg
  • σ(w) = σ(y+y+y+y) =
  • σ(y)+σ(y)+σ(y)+σ(y) =
  • 4σ(y) = 4·13² = 676kg²
  • σ(w) = √σ(w) = √676 = 26kg
                Resistência     Carga total

                σₓ = 10kg       σ(w) = 26kg

  • 280 - 300 = -20„

  • 676 + 100 = 776kg²
  • σ(T) = √σ(T) = √776 ≃ 27,9kg
