1 - Dividindo aleatoriamente um segmento em três partes, qual é a probabilidade de que esses novos segmentos formem um triângulo?

Inicie levantando algumas conjecturas para sua resolução.
  • ABC = A =1/3 / B = 1/3 / C = 1/3
  • Cada 1 teria 1/3 de chances para formar um triângulo.
  • Para se formar um triângulo a soma dos dois lados deve ser maior que o terceiro lado.
  • Identificado as opções para se formar um triângulo, pode-se calcular as probabilidades condicional para se formar um triângulo. Assim podemos determinar a probabilidade de formar um triângulo ao dividir aleatoriamente um segmento em três partes.
