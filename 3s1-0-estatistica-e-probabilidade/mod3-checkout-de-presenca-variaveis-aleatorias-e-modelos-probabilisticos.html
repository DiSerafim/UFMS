Pergunta 2

Seja a distribuição de probabilidades da variável aleatória X, assinale a alternativa que representa as condições sobre a e b para que g(x) seja uma função de probabilidade.

Figura 1: Distribuição de X

x -2 -1 0 1 2
g(x) -(a-b) b a a+b b-a
  • a. 0 <= a <= 1/3 e b = 1
  • b. 0 <= b <= 1/4 e a = 1/4
  • c. 0 <= a <= 1/4 e b = 1/4
  • d. a <= 1/2 e b = 1/3
  • e. 0 <= b <= 3/7 e a = 1/2
Resposta: b. 0 <= b <= 1/4 e a = 1/4. ❌
  • 0 <= b <= 1/4 e a = 1/4
  • g(-2) = -(a-b) = -(1/4 - b)
  • g(-1) = b
  • g(0) = a = 1/4
  • g(1) = a + b = 1/4 + b
  • g(2) = b - a = b - 1/4
  • Todas as probabilidades são não negativas para os valores de a e b especificados e a soma de todas as probabilidades é igual a 1.

Pergunta 3

Um dado de seis faces é jogado três vezes. A probabilidade de sair exatamente um 6 é 25/72.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Verdadeiro. ✅
  • Binomial: P(x = k) = (n/k)·pᵏ·(1-p)ⁿ⁻ᵏ
  • n = 3
  • k = 1
  • p = 1/6
  • P(x = 1) = (3/1)·(1/6)¹·(1 - 1/6)³⁻¹
  • P(x = 1) = 3 · 1/6 · (5/6)²
  • P(x = 1) = 3 · 1/6 · 25/36
  • P(x = 1) = 25/72

Pergunta 4

Os pesos dos fardos recebidos por um depósito têm média de 200kg e um desvio padrão de 30kg. Qual a probabilidade de 36 fardos recebidos ao acaso e carregados em um elevador não excederem o limite específico de segurança deste último, que é de 7668kg?

  • a. Entre 70% e 80%
  • b. Entre 90% e 100%
  • c. Entre 50% e 60%
  • d. Entre 80% e 90%
  • e. Entre 60% e 70%
Resposta: b. Entre 90% e 100%. ✅
  • Desvio padrão = Desvio padrão original/√n = 30/√36 = 5
  • Z = (7668 - 200 · 36)/5
  • Z = (7668 - 7200)/5
  • Z = 468/5
  • Z = 93,6

Pergunta 5

Chamadas telefônicas chegam à razão de 48 por hora no balcão de reservas da Catar Airlines. A probabilidade de chegarem 3 chamadas em um intervalo de tempo de cinco minutos é um valor entre 10% e 15%.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Falso. ✅
  • (λ) = 48/60 = 0.8
  • P(X = k) = (e⁻λ·λᵏ)/k!
  • P(X = 3) = (e⁻λ·λ³)/3!
  • P(X = 3) = (e⁻°,⁸ · 0,8³)/3!
  • P(X = 3) = (e⁻°,⁸ · 0,512)/6
  • P(X = 3) ≃ 0,0751 = 7,51%
