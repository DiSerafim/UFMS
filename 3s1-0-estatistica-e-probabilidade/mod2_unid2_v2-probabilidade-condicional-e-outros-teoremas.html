Exercício 1.

Contexto:

Numa determinada linha de produção sabe-se que 10% dos itens produzidos apresentam defeitos.

Um teste para verificar a ocorrência de defeitos será aplicado nesta linha de produção.

Este teste para detectar ou não a presença de defeitos garante 90% de acertos quando aplicado a itens efetivamente com defeitos e 80% de acertos quando aplicado a itens que não tem defeitos.

1. Qual o percentual de itens realmente com defeito dentre aqueles que o teste classificou como positivo(defeituoso)?

Resolução Gráfica.

                
                   o
                  /
                 /
            o---o---o
             \  
              \   o
               \ /
                o---o

Dados Iniciais

  • Sem defeito (0,9)
  • Com defeito (0,1)

Resultados dos testes

  • Sem defeito (0,9)... positivo(0,2) = 0,9 · 0,2 = P = 0,18
  • Sem defeito (0,9)... negativo(0,8) = 0,9 · 0,8 = P = 0,72
  • Com defeito (0,1)... positivo(0,9) = 0,1 · 0,9 = P = 0,09
  • Com defeito (0,1)... negativo(0,1) = 0,1 · 0,1 = P = 0,01
  • P = 0,18 + 0,72 + 0,09 + 0,01 = 1,0
  • P(Positivo) = 0,18 + 0,09 = 0,27
  • P(Realmente Defeituoso ∩ Positivo) = 0,09
  • P(Realmente Defeituoso/Positivo) = 0,09/0,27 = 33%

2. Qual o percentual de itens não defeituosos dentre aqueles que o teste classificou como negativo (não defeituosos)?

Resultado negativo

  • P = 0,18 + 0,72 + 0,09 + 0,01 = 1,0
  • P(Negativo) = 0,1 · 0,1 + 0,9 · 0,8 = 0,01 + 0,72 = 0,73
  • P(Negativo ∩ Não Defeito) = 0,9 · 0,8 = 0,72
  • P(Não Defeito/Negativo) = 0,72/0,73 = 98,6%
Exercício 2.

Contexto:

funcionamento de um sistema de resistores.

              ┌──[0,7]─┐         ┌──[0,85]─┐
            A-|--[0,7]-|--[0,9]--|         |-B
              └─-[0,7]─┘         └─-[0,85]─┘

Considerando a probabilidade de cada resistor estar funcionando, qual a chance do sistema funcionar(passar corrente entre A e B)?

Resolução:

P(V ou W) = P(V ∪ W) = 0,85 + 0,85 - 0,85 · 0,85 = 0,9775

            ┌──[V 0,85]─┐
            |           |-B = 97,75% de chances
            └─-[W 0,85]─┘

P(XYZ) = P(X ou Y ou z) = P(X ∪ Y ∪ Z) =

  • 0,7 + 0,7 + 0,7 - 0,7 · 0,7 - 0,7 · 0,7 - 0,7 · 0,7 + 0,7 · 0,7 · 0,7 = 0,973
              ┌──[X 0,7]─┐
            A-|--[Y 0,7]-| = 97,3%
              └─-[Z 0,7]─┘

P(sistema) = P(XYZ E K E VW) = P(XYZ ∩ K ∩ VW) = P(XYZ) · P(K) · P(VW)

  • 0,973 · 0,9 · 0,9775 = 0,855977 ≃ 86%
            ┌─ XYZ ----─┐         ┌─ VW -----─┐
            | ┌──[0,7]─┐| ┌─ K ─┐ |┌──[0,85]─┐|
            A-|--[0,7]-||-|[0,9]|-||         ||-B
            | └─-[0,7]─┘| └─---─┘ |└─-[0,85]─┘|

Exercício 3.

Contexto:

Problema das 3 portas.

  • O participante escolhe uma porta;
  • O apresentador abre uma das outras duas portas;
  • O participante deve trocar ou manter?

  • Porta 1 = 1/3 - Vazia

                 Carro/                 
                      \

  • Vazia = 2/3
  • Carro = 1/3

  • Porta 1 = 1/3 - Vazia
  • Porta 2 = 1/3 - Vazia

            Escolhe    Troca

  • Vazia = 1/3
