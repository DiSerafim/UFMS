Probabilidade Conceitos fundamentais

Objetivos:

  • Apresentar os conceitos principais sobre Probabilidade.
  • Apresentar as principais operações realizadas no contexto da Probabilidade.
Probabilidade

Consiste no estudo da aleatoriedade e da incerteza. Utiliza métodos para quantificação das chances associadas aos diversos resultados.

Fenômeno Determinístico ou Probabilístico.

Fenômeno Determinístico

  • O resultado é sempre o mesmo.

Fenômeno Probabilístico

  • O resultado é incerto e variável.

Probabilidade

  • Medida da informação sobre a ocorrência de um evento.

Estatística

  • Baseia-se no estudo dos fenômenos probabilísticos.
Fenômenos Aleatórios
  • Podem conduzir a diferentes resultados.
    • Exemplo:
    • Lançamento de moedas honestas.
    • Lançamento de dados.
    • Retiradas de cartas de baralhos
    • Determinação da vida útil de um componente eletrônico.
    • Características:
    • Os possíveis resultados são conhecidos (probabilidade)
    • O resultado final não é conhecido.
Experimento Aleatório
  • Características:
  • Cada experimento pode ser repetido indefinidamente sob as mesmas condições.
  • Quando o experimento for repetido um grande número de vezes e houver regularidade na explicação desse fenômeno, é possível estruturar um modelo matemático probabilístico.

Definição Frequencialista.

  • m: número de sucessos.
  • n: número de experimentos.
    •                 P = lim m/n
                        n → ∞

Definição Clássica.

  • P: Probabilidade.
  • M: número de sucessos.
  • N: número de possibilidades.
    •                 P = M/N
Espaço Amostral (S).
  • Espaço Amostral de um experimento aleatório é o conjunto de todos os possíveis resultados desse experimento.

    • Conjunto:

  • Uma coleção definida de objetos ou itens.

    • Ex: Lançamento de uma moeda.

    • S = { C, K }
    • É composto de 2 eventos

    Ex: Lançamento de duas moedas.

    • S = { (C, K), (C, C), (K, C), (K, K) }
    • É composto de 4 eventos

    Ex: Lançamento de um dado.

    • S = { 1, 2, 3, 4, 5, 6 }
    • É composto de 6 eventos
    Eventos Aleatórios (E).

    Definição

    • Qualquer subconjunto de um espaço amostral.
    • Resultado possível em experimentos aleatórios e que não é previsível.

    Ex: Lançamento de uma moeda.

    • S = { C, K }
    • E = K
    • P(E) = ?

    Ex: Lançamento de duas moedas.

    • S = { (C, K), (C, C), (K, C), (K, K) }
    • E = K, K
    • P(E) = ?

    Ex: Lançamento de um dado.

    • S = { 1, 2, 3, 4, 5, 6 }
    • E = 3
    • P(E) = ?

    Propriedades.

    • 0 ≤ P(E) ≤ 1
    • P(S) = 1
    Evento Complementar.
  • Formado por um único elemento do espaço amostral.

    • Evento Equiprovável.
  • Cada ponto amostral tem a mesma probabilidade de ocorrência.
  • P(1) = P(2) = P(3) = P(4) = P(5) = P(6)

    • Evento Mutuamente Excludentes.
  • S(A, B)

    • Operações entre Eventos.

    União - A ∪ B

    • Formado pelos pontos amostrais que pertencem a pelo menos um dos eventos.
    • Ocorre se A acorre OU B ocorre OU AMBOS ocorrem.

    Intersecção - A ∩ B

    • Formado pelos pontos amostrais que pertencem simultaneamente aos eventos A e B.
    • Ocorre apenas se A E B ocorrem.
    Axiomas da Probabilidade.

    1. Se ∅ é o conjunto vazio, então P(∅) = 0.

  • P(A ∩ B) = 0

    • 2. Teorema do Evento Complementar: se Ā é o complemento do evento A então:

  • P(Ā) = 1 - P(A)

    • 3. Teorema da Soma: se A e B são dois eventos quaisquer então:

  • P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

    • Se A e B são dois Eventos Mutuamente Excludentes

  • P(A ∪ B) = P(A) + P(B)

    • Se A, B e C são três eventos quaisquer então:

  • P(A ∪ B ∪ C) = P(A) + P(B) + P(C) - P(A ∩ B) - P(A ∩ C) - P(B ∩ C) + P(A ∩ B ∩ C)
