Exercício 1: Pesquisa de mercado:

Ferramenta para auxiliar no desenvolvimento de novos negócios, de novos produtos e serviços ou no aprimoramento de produtos e serviços já existentes.

Objetivo:

  • Avaliar o que os consumidores desejam e o grau de satisfação de um produto ou serviço.

Execução:

  • Perguntar aos consumidores, atuais ou potenciais, sua opinião a respeito de um produto ou serviço.

Caso:

Um empreendedor deseja lançar um novo produto e contrata uma empresa para realizar uma pesquisa de mercado com 500 pessoas. Entre as 500 pessoas, 157 manifestam interesse em comprar o produto.

Construa um intervalo de confiança com 92% para a probabilidade de uma pessoa adquirir o referido produto.

  • p̂ = 157/500 = 0,314
  • 1 - α = 92% → α = 8% → α/2 = 4% = 0,04
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5-a/2.
  • 0,5-a/2 = 0,5-0,04 = 0,46
  • Z(α/2) = 1,75.
            e₀ = z(α/2)√(p̂(1-p̂))/n
            e₀ = 1,75√(0,314(1-0,314))/2500
            e₀ = 0,036

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(0,314 - 0,036 ≤ π ≤ 0,314 + 0,036) = 1 - 0,08
  • P(0,278 ≤ π ≤ 0,350) = 0,92 = 92%

Conclusão:

Em 92% das vezes em que forem selecionadas 500 pessoas para avaliar o produto a proporção de pessoas interessadas em adquirir o produto se situará entre 0,278 e 0,350.

Exercício 2: Pesquisa de mercado II:

Caso:

Considere que i empreendedor do caso anterior deseje lançar seu produto no mercado apenas se as evidências trazidas pela pesquisa forem fortes o suficiente para garantir que ao menos 25% das pessoas tem interesse no produto.

Levando em conta as informações anteriores, o empreendedor deve lançar seu produto no mercado?

Considerando "99%" como um intervalo seguro:

  • 1 - α = 99% → α = 1% → α/2 = 0,5% = 0,005
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5-a/2.
  • 0,5-a/2 = 0,5-0,005 = 0,495
  • Z(α/2) = 2,575.
            e₀ = z(α/2)√(p̂(1-p̂))/n
            e₀ = 2,575√(0,314(1-0,314))/2500
            e₀ = 0,053

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(0,314 - 0,053 ≤ π ≤ 0,314 + 0,053) = 1 - 0,01
  • P(0,261 ≤ π ≤ 0,367) = 0,99 = 99%

Conclusão:

Como o intervalo de confiança considerando 1-α = 99% situa-se entre 26,1% e 36,7% o empreendedor pode considerar que ao menos 25% dos consumidores adquirirão seu produto em pelo menos 99% das ocasiões.

Exercício 3: Pesquisa Eleitoral:

Considere uma pesquisa eleitoral realizada em dois bairros, A e B.

No bairro A, um bairro central, foram entrevistados 500 eleitores, sendo que 100 deles declaram que iriam votar no Candidato X.

No bairro B, um bairro de periferia, dos 1.000 eleitores entrevistados, 300 declararam a intenção de votar no Candidato X.

a) Construa um intervalo de confiança para a intenção de voto no candidato X no bairro A, considerando um nível de confiança de 95%.

            e₀ = z(α/2)√(p̂(1-p̂))/n
            e₀ = 1,96√(0,2(1-0,2))/500
            e₀ = 0,035

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(0,2 - 0,035 ≤ π ≤ 0,2 + 0,035) = 1 - 0,05
  • P(0,165 ≤ π ≤ 0,235) = 0,95 = 95%

b) Faça o mesmo para o bairro B.

            e₀ = z(α/2)√(p̂(1-p̂))/n
            e₀ = 1,96√(0,3(1-0,3))/1000
            e₀ = 0,028

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(0,3 - 0,0,28 ≤ π ≤ 0,3 + 0,0,28) = 1 - 0,05
  • P(0,272 ≤ π ≤ 0,328) = 0,95 = 95%

c) Calcule um intervalo de confiança para a diferença entre as probabilidades de intenção de voto no candidato X nos bairros A e B, considerando = 5%.

  • W = A - B
  • W ~ N (PA - PB; (PA((1 - PA)/nA) + PB((1 - PB)/nB)))
  • p̂ = PA - PB = 0,2 - 0,3 = -0,1
            e₀ = z(α/2)√(PA(1-PA))/nA + (PB(1-PB))/nB
            e₀ = 1,96√(0,2(1-0,2))/500 + (0,3(1-0,3))/1000
            e₀ = 0,045

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(-0,1 - 0,045 ≤ π ≤ -0,1 + 0,045) = 1 - 0,01
  • P(0,145 ≤ π ≤ 0,055) = 0,95 = 95%

Resolução:

  • p̂A = 100/500 = 0,2.
  • p̂B = 300/1000 = 0,3.
  • 1 - α = 95% → α = 5% → α/2 = 2,5% = 0,025
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5-a/2.
  • 0,5-a/2 = 0,5-0,025 = 0,4750
  • Z(α/2) = 1,96.

Conclusão:

Como o intervalo de confiança considerando α = 95% situa-se entre -0,145% e -0,055% e não inclui o valor "0" então pode-se afirmar que existe diferença entre as intenções de voto no Bairro A e mo Bairro B.
