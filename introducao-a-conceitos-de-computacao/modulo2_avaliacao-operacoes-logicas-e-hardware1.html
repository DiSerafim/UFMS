Questão 1

Dada a expressão booleana a seguir (os símbolos ’, +, •, correspondem aos operadores NÃO, OU, E):
S = A • B • (C’ • D’ + C’ • D) + A’ + B + (C’ • D + C’ • D) + A • B’ • C’ • D

Qual o valor de S considerando as seguintes entradas: A=1, B=0, C=0, D=1?

Substituindo os valores A=1, B=0, C=0, D=1:
S = 1 • 0 • (0' • 1' + 0' • 1) + 1' + 0 + (0' • 1 + 0' • 1) + 1 • 1' • 0' • 1
S = 0 • (1 + 0) + 0 + 0 + (0 + 0) + 0
S = 0 • 1 + 0 + 0 + 0 + 0
S = 0 + 0 + 0 + 0
S = 0

Portanto, o valor de S considerando A=1, B=0, C=0, D=1 é 0. ❌
