Sistemas de numeração em computação.

→ Decimal: Base 10, usando dígitos de 0 a 9. É o sistema que usamos no dia a dia.
→ Binário: Base 2, utilizando apenas 0 e 1. Fundamental para computadores, pois a eletrônica digital trabalha com sinais binários.
→ Octal: Base 8, com dígitos de 0 a 7. Não tão comum atualmente, mas já foi usado em algumas aplicações.
→ Hexadecimal: Base 16, utilizando dígitos de 0 a 9 e letras de A a F para representar valores de 10 a 15. É útil na programação e representação de cores.

Sistema numérico na computação:
1. Binário (Base 2):
→ Usa apenas dois dígitos: 0 e 1.
→ Fundamental para a computação digital, onde a eletrônica manipula sinais binários.
→ Todos os dados são representados internamente em zeros e uns, desde textos até imagens e instruções de programas.

2. Decimal (Base 10):
→ O sistema numérico padrão que usamos no cotidiano.
→ Apesar de não ser o sistema fundamental para a computação, é usado em alguns contextos, especialmente na interação com usuários.

3. Octal (Base 8) e Hexadecimal (Base 16):
→ Octal usa 8 dígitos: 0 a 7.
→ Hexadecimal usa 16 dígitos: 0 a 9 e A a F (representando os valores de 10 a 15).
→ São usados na programação, pois fornecem uma maneira conveniente de representar números binários longos de maneira compacta e legível.

4. ASCII e Unicode:
→ Não são sistemas numéricos, mas são importantes para a computação.
→ ASCII (American Standard Code for Information Interchange) atribui valores numéricos a caracteres (letras, números, símbolos).
→ Unicode é uma codificação de caracteres mais abrangente, permitindo a representação de um conjunto muito maior de caracteres, incluindo vários idiomas e símbolos.

Conversão entre bases.

→ Decimal para Binário: Divisões sucessivas por 2 e coleta dos restos de cada divisão. Exemplo: 23 em binário é 10111. 

            23 ÷ 2 = 11 com resto 1
            11 ÷ 2 = 5 com resto 1
            5 ÷ 2 = 2 com resto 1
            2 ÷ 2 = 1 com resto 0
            1 ÷ 2 = 0 com resto 1

            Os restos lidos de baixo para cima são 10111, logo 23 em binário é 10111.
→ Binário para Decimal: Cada dígito binário é multiplicado pela potência de 2 correspondente à sua posição e somado. Exemplo: 10111 em decimal é 23. 
            1 * 2^4 + 0 * 2^3 + 1 * 2^2 + 1 * 2^1 + 1 * 2^0 = 16 + 0 + 4 + 2 + 1 = 23
→ Decimal para Hexadecimal e Octal: Primeiro converta para binário e depois agrupe os dígitos em conjuntos de 4 para hexadecimal ou de 3 para octal. 
            23 ÷ 2 = 11 com resto 1
            11 ÷ 2 = 5 com resto 1
            5 ÷ 2 = 2 com resto 1
            2 ÷ 2 = 1 com resto 0
            1 ÷ 2 = 0 com resto 1

            23 em binário é 10111.

            Agrupar em conjuntos de 4 (para hexadecimal):
            0010 1111

            Converter os grupos de 4 dígitos para hexadecimal:
            2 em hexadecimal é 2
            1111 em hexadecimal é F
            
            23 em hexadecimal é 17.
→ Hexadecimal/Octal para Decimal: Converta para binário e depois para decimal.
→ Hexadecimal para Binário e vice-versa: Use tabelas de conversão ou faça a conversão indireta passando por decimal.

Exemplos:
(1011)ₚ = 1·2³ + 0·2² + 1·2¹ + 1·2°
(1011)ₚ = 1·8 + 0·4 + 1·2 + 1·1 = 11

(230, 4)₈ = 2·8² + 3·8¹ + 0·8° + 4·8⁻¹

Codificação Binária - Números.

→ 1010 em binário é 10 em decimal (12^3 + 02^2 + 12^1 + 02^0). → 1111 em binário é 15 em decimal (12^3 + 12^2 + 12^1 + 12^0).

Exemplo:
Número binário: 10101

Para converter este número binário para decimal, usamos a seguinte fórmula:
(1 * 2^4) + (0 * 2^3) + (1 * 2^2) + (0 * 2^1) + (1 * 2^0)

Realizando as operações:
(1 * 16) + (0 * 8) + (1 * 4) + (0 * 2) + (1 * 1)
= 16 + 0 + 4 + 0 + 1
= 21
Portanto, o número binário 10101 é igual a 21 em decimal.
