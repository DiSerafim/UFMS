Ponto Flutuante.

Valores Reais: 

                            Mantissa ↓
            número = (sinal)0,d₁d₂d₃...dₘ x Bᴱˣᴾ
            -2,187 =    -   0,2187        x 10¹
Mantissa m: quantidade de dígitos na parte significativa do número.
Base B: conjunto de símbolos (ou algarismos).
Expoente exp: valor inteiro dentro de um intervalo[min, max].

Exemplo:
Uma calculadora opera na base 10 com 4 digitos na mantissa e expoente assumindo valores {-3, -2, -1, 0, 1, 2, 3}. 

                            ↓ Mantissa ↓
            numero  = (sinal)0,d₁d₂d₃d₄ x Bᴱˣᴾ
            -17,945 =    -   0, 1 7 9 4 x 10²
                    =    - 0,1794 x 10⁻²
                         TRUNCAMENTO!
        
            Maior Número
             (sinal)0,d₁d₂d₃d₄ x Bᴱˣᴾ
                +   0,9 9 9 9  x 10³ = 999,9
            
            Menor Número positivo
             0,d₁d₂d₃d₄ x Bᴱˣᴾ
             0,0 0 0 1  x 10° = 1x10⁻⁴

Valores Binários
Ex.:
Vamos representar -17,945 usando uma palavra com 16 bits e a seguinte estrutura: 

            | Sinal | Parte Inteira | Ponto | Parte Decimal |
            |   1   |     0111      |   0   | 1111011000    |
        
            Converter o valor absoluto para binário (17 para binário):
            17 ÷ 2 = 8 com resto 1
            8 ÷ 2 = 4 com resto 0
            4 ÷ 2 = 2 com resto 0
            2 ÷ 2 = 1 com resto 0
            1 ÷ 2 = 0 com resto 1

            Ao ler os restos de baixo para cima, temos: 10001.
            Inverter os bits para obter o complemento de um:
            10001 (binário original)
            01110 (complemento de um)

            Adicionar 1 para obter o complemento de dois:
            01110 (complemento de um)
            + 1
            -----
            01111 (complemento de dois)

adição e subtração de dois números binários de 4 bits.

            1001	    0100	    1101	        0101
            0110	    0010	    1000	        0100
