Software, Programa e Algoritmo

- Um algoritmo descreve a sequência de passos de um programa em uma linguagem próxima de um idioma.
Ex.: 

            Algoritmo para dividir dois valores numéricos.

            1 - Receba o dividendo.
            2 - Receba o divisor.
            3 - Se divisor vale zero:
                3.1 - Exibir menagem:
                    "Divisão por zero não é possível".
                3.2 - Terminar a execução.
            4 - Calcular a divisão:
                divisão = dividendo/divisor
            5 - Exibir mensagem "Resultado = divisão"
            6 - Terminar a execução.

- Um programa é uma sequência de instruções descritas por um algoritmo que permitem com que o computador realize uma tarefa.
Ex.: 

            # Programa para dividir dois valores numéricos

            # Passo 1: Receber o dividendo
            dividendo = float(input("Digite o dividendo: "))

            # Passo 2: Receber o divisor
            divisor = float(input("Digite o divisor: "))

            # Passo 3: Verificar se o divisor é zero
            if divisor == 0:
                print("Divisão por zero não é possível")
                exit()  # Terminar a execução do programa

            # Passo 4: Calcular a divisão
            resultado = dividendo / divisor

            # Passo 5: Exibir o resultado
            print("Resultado =", resultado)

            # Passo 6: Terminar a execução

- Um software é formado por um ou mais programas e estruturas de dados escritos para realizar alguma tarefa que resolva um problema ou torne mais fácil e segura a vida de pessoas ou de organizações.
Ex.: 

            Uma calculadora é um software que consiste em vários programas e estruturas de dados para realizar várias operações matemáticas, incluindo a divisão.

            Imagine um software de calculadora que inclui a função de divisão juntamente com outras operações matemáticas, como adição, subtração e multiplicação.
            Este software teria uma interface onde o usuário pode inserir os números e selecionar a operação desejada.
            Cada operação seria realizada por um programa específico dentro do software.

Tipos de Software

Software básico ou de sistema:
- Coleção de programas escritos para apoiar outros programas. Possui contato mais direto com o hardware, gerenciando a execução de outro software.
Ex.: 

            Sistema Operacional Windows: Desenvolvido pela Microsoft, é um dos sistemas operacionais mais populares para PCs e laptops.

            Sistema Operacional macOS: Desenvolvido pela Apple, é o sistema operacional usado nos computadores Mac.

            Sistema Operacional Linux: Existem várias distribuições de Linux, como Ubuntu, Fedora e Debian, que são sistemas operacionais de código aberto utilizados em computadores pessoais, servidores e dispositivos integrados.

            BIOS (Basic Input/Output System): É um software básico que inicializa o hardware durante a inicialização de um computador e fornece a interface entre o hardware e o sistema operacional.

            Firmware de Dispositivos: Isso inclui firmware embutido em dispositivos como roteadores, impressoras, câmeras digitais, entre outros, que controla seu funcionamento básico.

            Drivers de Dispositivos: São programas que permitem que o sistema operacional comunique-se com hardware específico, como placas de vídeo, impressoras, teclados, etc.

Algoritmos

Conjunto finito de regras, bem definido (sem ambiguidades), voltado à solução de um problema em tempo finito. Deve possuir 3 qualidades: 1. Cada passo do algoritmo deve ser uma instrução que possa ser realizada.
2. A ordem dos passos deve ser precisamente determinada.
3. O algoritmo deve ter fim.

Dois aspectos devem ser levados em consideração no projeto de algoritmos:
Corretude:
- Para cada entrada possível, o algoritmo terá fim e produzirá a resposta desejada;
Eficiência:
- A quantidade de recursos (tempo e armazenamento) sonsumidos na execução do algoritmo.
Ex.: 

            Algoritmo para Troca de Pneu de um Carro:

            Reúna as Ferramentas:
            - Certifique-se de ter uma chave de roda, macaco, chave de porca e estepe disponíveis.

            Estacione o Carro em Local Seguro:
            - Escolha um local plano, longe do tráfego, e acione o freio de estacionamento.

            Ligue as Luzes de Emergência:
            - Para alertar outros motoristas sobre a situação.
            
            Retire a Tampa da Roda e Desaperte as Porcas:
            - Use a chave de roda para soltar as porcas da roda antes de levantar o carro.
            
            Posicione o Macaco:
            - Encontre o ponto de elevação recomendado no chassi do carro e posicione o macaco.
            
            Levante o Carro:
            - Use o macaco para levantar o carro do chão até que a roda esteja desobstruída.
            
            Remova as Porcas e a Roda Danificada:
            - Utilize a chave de roda para remover completamente as porcas e retire a roda danificada.
            
            Coloque a Estepe:
            - Posicione a estepe no lugar, alinhando os furos com os pinos de fixação.
            
            Aperte as Porcas Manualmente:
            - Aperte as porcas manualmente para fixar a roda no lugar.
            
            Baixe o Carro com Cuidado:
            - Use o macaco para abaixar lentamente o carro até o chão.
            
            Aperte as Porcas com a Chave de Roda:
            - Aperte as porcas completamente com a chave de roda.
            
            Substitua a Tampa da Roda:
            - Coloque a tampa na roda substituída.
            
            Guarde as Ferramentas:
            - Guarde todas as ferramentas usadas no porta-malas.
            
            Verifique a Pressão do Pneu Substituído:
            - Use um medidor de pressão para garantir que o pneu substituído tenha a pressão adequada.

Linguagem de Programação

- Permite descrever, de forma compreensível para o computador, os passos a serem por ele realizados na execução de uma tarefa.
- Toda linguagem de programação possui um conjunto limitado de símbolos, estruturas de dados e comandos utilizados para criar os programas.

Linguagem de máquina - primeira geração.
Linguagem de montagem - segunda geração.
Linguagens de auto nível - terceira geração.
Linguagens de muito auto nível - quarta geração.

- Funcional. Ex.: 
Haskell, Lisp, Scala.
- Lógico.Ex.: 
Prolog, Datalog.
- Orientado a objetos. Ex.: 
Java, Python, C++

Ciclo de Vida do Software.

Requisitos:
- Compreensão e documentação dos requisitos do software, ou seja, o que o software deve fazer e quais problemas deve resolver.

Análise:
- Análise detalhada dos requisitos para identificar possíveis desafios, riscos e soluções. Isso pode incluir modelagem de dados e processos.

Projeto:
- Criação da arquitetura do software com base nos requisitos e análises. Isso inclui decisões sobre linguagem de programação, estrutura de dados, interfaces de usuário, etc.

Implementação/Desenvolvimento:
- O desenvolvimento real do software acontece nesta fase. Os programadores codificam o software conforme as especificações do projeto.

Testes:
- Verificação e validação do software para garantir que ele atenda aos requisitos. Isso inclui testes de unidade, integração, sistema e aceitação do usuário.

Implantação/Instalação:
- O software é disponibilizado ao usuário final. Pode envolver a instalação em sistemas específicos, treinamento de usuários e configurações adicionais.

Manutenção:
- Após a implantação, o software pode passar por manutenções corretivas, evolutivas ou adaptativas para corrigir bugs, adicionar novos recursos ou ajustar a funcionalidade conforme necessário.

Aposentadoria/Descontinuação:
- Em algum momento, o software pode ser retirado de uso devido a obsolescência, mudanças de requisitos ou por ser substituído por versões mais atualizadas.
