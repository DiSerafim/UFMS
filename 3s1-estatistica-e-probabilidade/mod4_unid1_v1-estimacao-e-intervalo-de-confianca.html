Exercício 1:

Caso: Notas da Prova.

A nota da prova numa determinada prova de estatística é tal que seu desvio padrão é conhecido e igual a 5,0 pontos

Uma amostra de 100 provas realizadas pelos alunos foi selecionada aleatoriamente e obteve-se nota média amostral de 7,0 pontos.

Deseja-se construir um intervalo de confiança para a verdadeira nota média desta prova com nível de 95% de confiança

1 - α = 95% → α = 5% → α/2 = 2,5% = 0,025

  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5 - a/2
  • 0,5 - a/2 = 0,5 - 0,025 = 0,4750
  • Z(α/2) = 1,96
  • ...
  • (α/2) = 2,5%
  • Z(α/2) = 1,96
  • Contexto:
  • e₀ = Z(α/2) σ/√n
  • e₀ = 1,96·5/√100 = 0,98
  • Intervalo de confiança:
  • p(x̄ - e₀ ≤ μ ≤ x̄ + e₀) = 1 - α
  • p(7 - 0,98 ≤ μ ≤ 7 + 0,98) = 1 - 0,05
  • p(6,02 ≤ μ ≤ 7,98) = 0,95 = 95%
  • Conclusão:
  • A nota média obtida se situará entre 6,02 e 7,98 pontos.
Exercício 2:

Caso: Número de Residentes.

Uma pesquisa con intuito de conhecer a realidade dos alunos da rede de ensino registrou que 3,2 pessoas residiam em média entre os 18 diferentes domicílios pesquisados nesta amostra selecionada aleatoriamente.

O IBGE possui registro de que desvio padrão da população associava à variável "número de residentes por domicílio" é igual a "2 residentes por domicílio".

a) Elabore um intervalo de confiança para o número de residentes por domicílio desta população considerando nível de 98% de confiança.

  • 1 - α = 98% → α = 2% → α/2 = 1% = 0,01
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5 - α/2
  • 0,5 - α/2 = 0,5 - 0,01 = 0,49
  • Z(α/2) ≃ 2,33
  • Contexto:
  • e₀ = Z(α/2) σ/√n
  • e₀ = 2,33·2/√18 = 1,09
  • Intervalo de confiança:
  • p(x̄ - e₀ ≤ μ ≤ x̄ + e₀) = 1 - α
  • p(3,2 - 1,09 ≤ μ ≤ 3,2 + 1,09) = 1 - 0,02
  • p(2,11 ≤ μ ≤ 4,29) = 98%
  • Conclusão:
  • Média de 2,11 a 4,29.

b) Elabore um intervalo de confiança para o número de residentes por domicílio desta população considerando nível de 90% de confiança.

  • 1 - α = 90% → α = 10% → α/2 = 5% = 0,05
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5 - α/2
  • 0,5 - α/2 = 0,5 - 0,05 = 0,45
  • Z(α/2) ≃ 1,645
  • Contexto:
  • e₀ = Z(α/2) σ/√n
  • e₀ = 1,645·2/√18 = 0,77
  • Intervalo de confiança:
  • p(x̄ - e₀ ≤ μ ≤ x̄ + e₀) = 1 - α
  • p(3,2 - 0,77 ≤ μ ≤ 3,2 + 0,77) = 1 - 0,1
  • p(2,43 ≤ μ ≤ 3,97) = 90%
  • Conclusão:
  • a) Para 98%: 2,11 a 4,29 residentes.
  • b) Para 90%: 2,43 a 3,97 residentes.
Exercício 3:

Caso: Quantidade de açúcar.

Considere uma padaria que produz doces com média de açúcar de 100 mg/doce e um desvio padrão de 10 mg/doce.

A distribuição desses valores nessa produção da padaria tem distribuição normal.

Encontre a probabilidade de uma amostra aleatória de n = 5 doces, retirada aleatoriamente pela equipe de controle da padaria, ter uma média de açúcar menor que 97 mg/doce.

  • Média:
  • μ₍ₓ₎ = 100 mg/pão
  • Variância:
  • σ₍ₓ₎ = σ/n = 10/5 = 2 mg/pão
  • Fórmula:
  • Z₍ᵃ⁄₂₎ = (x̄ - μ)/(σ/n)
  • Z = (x̄ - μ)/(σ₍ₓ₎) = (97-100)/2 = -1,5.
  • Na tabela normal busca-se o valor de |Z| = 1,50:
  • P(z) = 0,4332
  • Z = -1,50
  • Intervalo de confiança:
  • p(x < 97) = p(x < -1,50) = 0,5 - 0,4332 = 0,0668
  • p(x < 97) = 6,68%
  • Conclusão:
  • A probabilidade da amosta contendo 5 doces ter média amostral inferior a 97 mg/doce é de 6,68%.
