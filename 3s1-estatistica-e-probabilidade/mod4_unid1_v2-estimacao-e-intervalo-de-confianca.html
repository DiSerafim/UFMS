Exercício 1:

Caso: Linha de Montagem.

  • Criada por Henry Ford.
  • Nova forma de produzir.
  • Maior produtividade.
  • Menor custo de produção.

Estudos detalhados indicam que a variância do tempo de produção de um veículo é conhecida e igual a 16 dias².

Uma amostra de 64 veículos produzidos foi selecionada aleatoriamente e obteve-se tempo médio de produção de 5,0 dias.

Deseja-se construir um intervalo de confiança para o verdadeiro tempo médio de produção com nível de 95% de confiança.

  • Nível de confiança:
  • 1 - α = 95% → α = 5% → α/2 = 2,5% = 0,025
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5 - a/2
  • 0,5 - a/2 = 0,5 - 0,025 = 0,4750
  • Z(α/2) = 1,96
  • ...
  • (α/2) = 2,5%
  • Z(α/2) = 1,96
  • Contexto:
  • e₀ = Z(α/2)·√σ/n
  • e₀ = 1,96·√16/64 = 0,98
  • Intervalo de confiança:
  • p(x̄ - e₀ ≤ μ ≤ x̄ + e₀) = 1 - α
  • p(5 - 0,98 ≤ μ ≤ 5 + 0,98) = 1 - 0,05
  • p(4,02 ≤ μ ≤ 5,98) = 0,95 = 95%
  • Conclusão:
  • Em 95% das vezes em que forem selecionadas amostras de 64 veículos produzidos o tempo médio de produção se situará entre 4,02 e 5,98 dias.
Exercício 2:

Caso: Medições com Multímetros.

Aparelhos de medição bastante populares.

Multímetros analógicos utilizam galvanômetro, bobina elétrica montada em um anel ao redor de um imã.

Indústria Brasileira que projeta, desenvolve e fabrica equipamentos de teste de medição.

Historicamente o desvio padrão é de 3,0 μV

Lote de 18 amostras para verificação da qualidade resultou em medição de 4,7 μV

a) Elabore um intervalo de confiança para sensitividade média desta população considerando nível de 98% de confiança.

  • Nível de confiança:
  • 1 - α = 98% → α = 2% → α/2 = 1% = 0,01
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5 - a/2
  • 0,5 - α/2 = 0,5 - 0,01 = 0,49
  • Z(α/2) ≃ 2,33
  • ...
  • (α/2) = 1%
  • Z(α/2) = 2,33
  • Contexto (Intervalo):
  • e₀ = Z(α/2)·σ/√n
  • e₀ = 2,33·3√18 = 1,65
  • Intervalo de confiança (Fórmula):
  • p(x̄ - e₀ ≤ μ ≤ x̄ + e₀) = 1 - α
  • p(4,7 - 1,65 ≤ μ ≤ 4,7 + 1,65) = 1 - 0,02
  • p(3,5 ≤ μ ≤ 6,35) = 0,98 = 98%
  • Conclusão:
  • Para 98% (3,5μV e 6,35μV) medição.

b) Elabore um intervalo de confiança para sensitividade média desta população considerando nível de 90% de confiança.

  • Nível de confiança:
  • 1 - α = 90% → α = 10% → α/2 = 5% = 0,05
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5 - a/2
  • 0,5 - α/2 = 0,5 - 0,05 = 0,45
  • Z(α/2) ≃ 1,645
  • ...
  • (α/2) = 1%
  • Z(α/2) = 1,645
  • Contexto (Intervalo):
  • e₀ = Z(α/2)·σ/√n
  • e₀ = 1,645·3√18 = 1,16
  • Intervalo de confiança (Fórmula):
  • p(x̄ - e₀ ≤ μ ≤ x̄ + e₀) = 1 - α
  • p(4,7 - 1,65 ≤ μ ≤ 4,7 + 1,65) = 1 - 0,1
  • p(3,54 ≤ μ ≤ 5,86) = 0,90 = 90%
  • Conclusão:
  • Para 90% (3,54μV e 5,86μV) medição.
