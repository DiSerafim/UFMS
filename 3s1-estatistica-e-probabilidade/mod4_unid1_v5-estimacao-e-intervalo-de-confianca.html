Caso: Teste de Vazamento

Considere testes de vazamento realizados em dois municípios, A e B, buscando identificar o percentual de edifícios que possuem alguma falha na tubulação.

No município A foram examinadas 500 edifícios, sendo que 100 deles apresentam falhas na tubulação. No município B dos 1.000 edifícios examinadas, 300 apresentaram falhas.

Resolução:

  • p̂A = 100/500 = 0,2.
  • p̂B = 300/1000 = 0,3.
  • 1 - α = 95% → α = 5% → α/2 = 2,5% = 0,025
  • Z(α/2) = ?
  • Na tabela normal busca-se o valor de 0,5-a/2.
  • 0,5-a/2 = 0,5-0,025 = 0,4750
  • Z(α/2) = 1,96.

a) Construa um intervalo de confiança para o percentual de edifícios com falhas na tubulação do município A considerando um nível de confiança de 95%.

            e₀ = z(α/2)√(p̂(1-p̂))/n
            e₀ = 1,96√(0,2(1-0,2))/500
            e₀ = 0,035

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(0,2 - 0,035 ≤ π ≤ 0,2 + 0,035) = 1 - 0,05
  • P(0,165 ≤ π ≤ 0,235) = 0,95 = 95%

b) Faça o mesmo para o município B.

            e₀ = z(α/2)√(p̂(1-p̂))/n
            e₀ = 1,96√(0,3(1-0,3))/1000
            e₀ = 0,028

Intervalo de confiança:

  • P(p̂ - e₀ ≤ π ≤ p̂ + e₀) = 1-α
  • P(0,3 - 0,0,28 ≤ π ≤ 0,3 + 0,0,28) = 1 - 0,05
  • P(0,272 ≤ π ≤ 0,328) = 0,95 = 95%
