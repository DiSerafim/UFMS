Amostragem

Métodos Estatísticos

Coleta, organização, apresentação, análise e interpretação de dados experimentais.

Seu objetivo fundamental é o estudo dos parâmetros de uma população.

Amostragem: duas abordagens. Censo:

Investigação de todos os elementos.

Amostragem:

seleção de alguns elementos da população - pesquisa em uma amostra.

Conceitos

Pesquisas podem ser realizadas por meio do estudo dos elementos de uma amostra.

A amostra é extraída da população que se pretende analisar.

Amostragem é o processo de escolha da amostra.

Amostragem é a atividade inicial em qualquer estudo estatístico.

Muitos especialistas consideram a etapa mais importante.

Consiste na escolha criteriosa dos elementos que participarão do estudo.

Exemplo: Pesquisa Eleitorais.

Realizadas para conhecer as tendências do eleitorado.

Resultados representativos são obtidos a partir de dados obtidos num conjunto de pessoas representativas.

Representatividade:

Características socioeconômicas, culturais, religiosas, etc. Tão próximas quanto possível da população cujos resultados da pesquisa serão a ela estendidos.

Etapas da pesquisa:

Escolha da amostra, redação do questionário, entrevista, codificação dos dados e apuração dos resultados

Registradas no TSE:

Sistema de Registro de Pesquisas Eleitorais.

Amostragem e Amostra:

Amostragem - processo de coleta das informações de uma parte de população, a amostra.

Amostra - parte ou subconjunto finito representativo de uma população selecionada segundo métodos adequados.

Método Estatístico:

1. Definição do problema;

1.1 identificação da população que será pesquisada.

2. Planejamento da pesquisa;

2.1 quais procedimentos para se resolver o problema?

2.2 qual o método de seleção da amostra?

2.3 como levantar informações da amostra?

2.4 que dados deverão ser obtidos?

2.5 qual o cronograma de atividades?

2.6 quais os custos envolvidos?

3. Coleta dos dados;

3.1 contratação e treinamento dos pesquisadores.

3.2 encontro com os participantes da pesquisa (membros da amostra).

3.3 aplicação dos questionários.

3.4 coleta das respostas.

4. Apuração dos dados;

4.1 tabulação das respostas da amostra.

4.2 consolidação de acordo com os critérios desejados.

5. Apresentação dos dados;

5.1 formatação das tabelas.

5.2 formatação dos gráficos.

6. Análise e interpretação dos dados;

Inferência Estatística:

Generalização dos resultados obtidos a partir da amostra para toda uma população.

Deve considerar as estatísticas de teste.

Deve considerar as condições de erro previstas.

Pode ser mais efetivo que entrevistar toda a população.

Amostra não probabilística

Amostragens intencionais.

Escolha deliberada dos elementos da amostra.

as amostras não probabilísticas não garantem a representatividade da população.

Resultados não são generalizáveis.

Tipos

1. Amostragem Acidental;

1.1. Formada por elementos conforme estes vão aparecendo.

1.2. Elementos são os possíveis de se obter.

1.3. Encerra quando completar o número de elementos da amostra.

1.4. Exemplo: pesquisa de opinião em que os entrevistados são acidentalmente escolhidos.

2. Amostragem Intencional;

2.1. Formada por elementos escolhidos por determinado critério.

2.2. Grupo de elementos intencionalmente escolhidos para compor a amostra.

2.3. Exemplo: pesquisa de opinião em que os entrevistados são especialistas em determinado assunto.

3. Amostragem por Quotas;

3.1. Classificação da população de acordo com propriedades.

3.2. Determinação da proporção da população para cada propriedade (quotas).

3.3. Seleção não aleatórias de elementos de cada quota identificada.

Amostra probabilística

Cada elemento da população possui probabilidade conhecida de ser escolhido.

Usualmente a probabilidade é a mesma entre os elementos.

População com N elementos.

P(ₚₐᵣₜᵢᶜᵢₚₐᵣ) = 1/N

Pode-se realizar inferências sobre a população a partir dos parâmetros estudados na amostra.

Resultados são generalizáveis.

Tipos

1. Amostragem Aleatória Simples;

1.1. Seleção de amostras de tamanho "K" dentre as "N" unidades da população.

1.2. Amostragem realizada sem reposição.

1.3. Número de amostras possíveis:

1.3.1. (n/k) = n! / k!(n-k)!

1.4. Amostra pode ser escolhida por diversos métodos.

1.5. Ex: Tabelas de Números Aleatórios (TNA) ou computadores para gerar números aleatórios.

1.5.1. As unidades da população são numeradas de 1 a N.

1.5.2. Escolhe-se K números compreendidos entre 1 e M.

1.5.3. Equivale a um sorteiro com todos os números misturados numa urna.

1.6. Na prática é escolhida unidade por unidade.

2. Amostragem Aleatória Estratificada;

2.1. Obtida separando-se as unidades da população em grupos não superpostos chamados estratos.

2.2. Seleciona-se amostras aleatórias independentes simples de cada estrato.

2.3. Dois tipos de amostragem estratificada.

2.3.1. De igual tamanho.

2.3.2. Proporcional.

2.3.3. N → Nº de unidades da população.

2.3.4. n → Nº de unidades das amostras.

2.3.5. Nₐ → Nº de unidades do estrato A.

2.3.6. nₐ → Nº de unidades de A.

2.3.7. Nₐ/N = nₐ/n → nₐ = (n/N)·Nₐ

3. Amostragem Sistemática;

3.1. Elementos da população estão ordenados.

3.2. A seleção dos elementos que constituirão a amostra obedece um determinado intervalo.

3.3. A amostra sistemática de n elementos de uma população de tamanho N - K deve ser menor ou igual a N/n.

3.4. Fácil de executar - menos sujeita a erros do entrevistador.

3.5. Frequentemente proporciona mais informações por custo unitário do que a aleatória simples.

3.6. Estabelecer o intervalo de amostragem K:

3.6.1. k = N/n

3.6.2. Para valores de K=N/n, arredonda para o valor inteiro menor.

3.7. Iniciar aleatoriamente a composição da amostra.

3.7.1. b → inicio.

3.7.2. (n° de ordem inicial sorteado na TNA).

3.7.3. 0 < b < k

3.7.4. Diretrizes para calcular as amostras:

3.7.5. Composição da amostra:

3.7.6. Item → b

3.7.7. Item → b + k.

3.7.8. Item → b + 2k.

4. Amostragem por Conglomerado;

4.1. Amostra aleatória simples em que cada unidade de amostragem é um grupo ou conglomerado de elementos.

4.2. Necessário especificar conglomerados apropriados.

4.3. Todos os elementos do conglomerado fazem parte da amostra.

4.4. Os elementos em um conglomerado tendem a ter características similares.

4.5. O numero de elementos num conglomerado deve ser pequeno.

4.6. O número de conglomerados deve ser razoavelmente grande.

4.7. Exemplo: levantamento da população de uma cidade.

4.7.1. Mapa indicando cada quarteirão.

4.7.2. Seleciona amostra dos quarteirões.

4.7.3. Realiza contagem completa de todos os que residem nos quarteirões sorteados.
