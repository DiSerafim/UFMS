Pergunta 1

Na programação orientada a objetos, o que são exceções?

  • a. São avisos emitidos pelo compilador.
  • b. São erros que ocorrem durante a execução do programa.
  • c. São erros de compilação.
  • d. São erros de sintaxe.
Resposta: b. ✅

Resolução:

  • São eventos que ocorrem durante a execução de um programa que interrompem o fluxo normal de execução. Esses eventos podem ser erros, como divisão por zero ou acesso a um elemento de um array fora dos limites, ou situações excepcionais que podem ocorrer durante a execução do programa, como falhas na comunicação com um banco de dados ou falhas na leitura de um arquivo. As exceções são usadas para lidar com esses eventos de forma estruturada e controlada, permitindo que o programa reaja adequadamente a eles..

    Pergunta 3

    Indique a opção que mostre qual é a diferença entre um FileInputStream e um FileReader em Java:

    • a. FileInputStream é utilizado para leitura de arquivos binários, enquanto FileReader é utilizado para leitura de arquivos de texto.
    • b. FileInputStream e FileReader são utilizados para a mesma finalidade, não há diferença entre eles.
    • c. FileInputStream e FileReader são objetos equivalentes, não há diferença entre eles.
    • d. FileInputStream é utilizado para leitura de arquivos de texto, enquanto FileReader é utilizado para leitura de arquivos binários.
    Resposta: a. ✅

    Resolução:

  • Essa é a diferença fundamental entre FileInputStream e FileReader em Java. O FileInputStream é usado para ler dados brutos de arquivos, que podem ser qualquer tipo de arquivo, incluindo binários. Por outro lado, o FileReader é otimizado para ler arquivos de texto, realizando automaticamente a decodificação dos bytes em caracteres de acordo com o conjunto de caracteres padrão do sistema.

    Pergunta 8

    O que são coleções em Java?

    • a. São classes que representam tipos de dados genéricos.
    • b. São estruturas de dados que permitem armazenar e manipular um grupo de objetos.
    • c. São bibliotecas que não fazem parte da API padrão do Java.
    • d. São interfaces que definem um conjunto de métodos para manipular um grupo de objetos.
    Resposta: b. ✅

    Resolução:

Elas fornecem implementações de diversas estruturas, como listas, conjuntos, mapas, entre outros, para atender diferentes necessidades de armazenamento e manipulação de dados.

    • Pergunta 9

    Qual é a função de um bloco try-catch em Java?.

    • a. É um b'loco de código que permite capturar e tratar exceções em Java.
    • b. É um 'bloco de código que é executado independentemente da ocorrência de exceções.
    • c. É um bloco de código que é executado antes de ocorrer uma exceção.
    • d. É um bloco de código que é executado apenas se ocorrer uma exceção.
    Resposta: a. ✅

    Resolução:

    Um bloco try-catch em Java é usado para capturar exceções que podem ocorrer durante a execução do código dentro do bloco try. Se uma exceção é lançada dentro do bloco try, ela é capturada e tratada pelos blocos catch correspondentes, permitindo que o programa lide com a exceção de uma maneira controlada.
