    • Pergunta 2

    Qual é o valor de x e de y de acordo com o código abaixo?

    • int x = 4;
    • int y = ++x + 100;
    • a. x = 4 e y = 105
    • b. x = 5 e y = 105
    • c. x = 5 e y = 104
    • d. x = 4 e y = 104
    Resposta: b. x = 5 e y = 105 ✅

    Resolução:

  • Isso ocorre porque o operador de pré-incremento (++x) primeiro incrementa o valor de x para 5 e, em seguida, atribui o resultado (5) a y, que é então somado com 100. Então, x é 5 e y é 105.

    • Pergunta 5

    Qual foi a estratégia adotada pelos projetistas da linguagem Java para torná-la independente de plataforma?

    • a. Interpretar o código com a interpretadora da máquina virtual Java (JVM).
    • b. Java usa o mesmo compilador de C++. Ela faz isso para ter uma execução mais rápida. A interpretação usa o interpretador da linguagem Python pelo mesmo motivo.
    • c. Compilar o código para um código intermediário (bytecode) e depois a máquina virtual Java (JVM) interpreta o bytecode.
    • d. Java não é independente de plataforma.
    Resposta: c. Compilar o código para um código intermediário (bytecode) e depois a máquina virtual Java (JVM) interpreta o bytecode. ✅

    Resolução:

  • Ao compilar um programa Java, o código-fonte é convertido em bytecode, que é uma representação intermediária independente de plataforma. Esse bytecode pode ser executado em qualquer máquina que tenha uma JVM (Java Virtual Machine), proporcionando portabilidade ao código Java.

    • Pergunta 7

    O que é impresso após a execução do código?

    • System.out.println(1+2+""+2+1);
    • a. 321
    • b. 1221
    • c. Não compila
    • d. 6
    Resposta: a. 321. ✅

    Resolução:

    1. Primeiro, ocorre a adição de 1 e 2, que resulta em 3.
    2. Em seguida, a expressão "" é concatenada com o resultado anterior (3). Quando você concatena uma string vazia "" com um número, o número é implicitamente convertido em uma string. Portanto, "3" é a string resultante.
    3. Depois, essa string "3" é concatenada com o número 2, resultando na string "32".
    4. Por fim, a string "32" é concatenada com o número 1, resultando na string "321".

    Considere o comando que insere uma quebra de linha ao final da impressão.

    Resposta: System.out.println("TESTE"); ✅

    • Pergunta 9

    Associe os caracteres de cada operador de acordo com o nome do operador correspondente.

    • Operador ternário (...)
    • Operador unário de decremento (...)
    • Operador de módulo (...)
    • Operador unário de incremento (...)
    Resposta: . ✅

    Resolução:

    • Operador ternário: ?
      • é usado para criar expressões condicionais. Ele avalia uma expressão booleana e, dependendo do resultado dessa avaliação, retorna um dos dois valores. Por exemplo, em condição ? valor_se_verdadeiro : valor_se_falso, se a condição for verdadeira, o valor_se_verdadeiro é retornado; caso contrário, o valor_se_falso é retornado.
    • Operador unário de decremento: --
      • é usado para decrementar o valor de uma variável em uma unidade. Por exemplo, x-- decrementará o valor de x em 1 após a avaliação da expressão em que x é usado.
    • Operador de módulo: %
      • retorna o resto da divisão entre dois números. Por exemplo, 10 % 3 retornará 1, pois 10 dividido por 3 é igual a 3, com um resto de 1.
    • Operador unário de incremento: ++
      • é usado para incrementar o valor de uma variável em uma unidade. Por exemplo, x++ incrementará o valor de x em 1 após a avaliação da expressão em que x é usado.
