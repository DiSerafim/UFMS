Teoremas importantes e suas interpretações geométricas

Teorema do Máximo e Mínimo Absolutos para Funções Contínuas
Seja f uma função contínua em 

[a, b] ⊂ ℝ
Então existe pontos 
c, d ∈ [a, b]
tal que: 
            f(d) <= f(x) <= f(c), ∀x ∈ [a, b]

            d é o ponto mínimo absoluto
            c é o ponto máximo absoluto
Função Limitada
Uma consequência imediata é que se f é contínua em 
[a, b]
então ela é limitada, isto é 
M > 0
tal que: 
            |f(x)| <= M, ∀x ∈ [a, b]
            f(x) = 1/x
            limitada em
            [a, b] se 0 ∉ [a,b]

Teorema de Rolle
Seja f uma função contínua 

[a, b] ⊂ ℝ
e derivável em (a, b). Se 
f(a) = f(b)
então 
∃c ∈ (a, b) tal que f'(c) = 0
Obs: Podem existir mais de um ponto satisfazendo o teorema Teorema de Rolle: Prova
Caso 1 
            f(x) = k, ∀x ∈ [a, b]
            f'(x) = 0, ∀x ∈ (a, b)
Qualquer 
c ∈ (a, b)
satisfaz o teorema. Caso 2 
            ∃x ∈ (a, b), com f(x) > f(a)
f tem máximo absoluto em [a, b], mas 
f(a) = f(b)
daí este máximo está em algum ponto 
c ∈ (a, b)
ou seja, é um máximo local e daí 
f'(c) = 0

Teorema do Valor Médio para Derivados (TVM)
Seja f uma função contínua em 

[a, b] ⊂ ℝ
e derivável em (a, b). Então 
∃c ∈ (a, b) tal que f(c) - f(a)/b -a = f'(c)
Teorema do Valor Médio: Prova
Reta passando pelos pontos: 
            (a, f(a)) e (b, f(b))
            g(x) = f(a) + (f(b)-f(a)/b-a)·(x - a)
Considere a função 
            h(x) = f(x)-g(x) = f(x)-f(a)-(f(b)-f(a)/b-a)·(x-a) 
            h(a) = h(b) = 0     - Pelo Teorema de Rolle
            ∃c ∈ (a, b) com h'(c) = 0
            h'(x) = f'(x)-(f(b)-f(a)/b-a)

            Substituindo em x = c
            0 = h'(c) = f'(c) - (f(b)-f(a)/b-a)
            e daí
            f'(c) = (f(b)-f(a))/b-a
Exemplo 1: 
            Se f'(x) é limitada em [a, b] ⊂ ℝ então, pelo TVM, para quaisquer pontos x1, x2 ∈ [a, b]

            ∃c ∈ (x₁, x₂) tal que
            |f(x₂)-f(x₁)| = |f'(c)||x₂ - x₁|
            como f'(x) é limitada
            ∃M > 0 ⇒ |f'(x)| < M, ∀x ∈ [a, b] Daí
            |f'(c)| < M ⇒ |f(x₂)-f(x₁)| < M |x₂ - x₁|
Exemplo 2: 
            Teorema: f'(x) = 0, ∀x ∈ (a, b) ⇒ f(x) = k = const
            Prova: x₁, x₂ ∈ (a, b) com x₁ < x₂

            Como f é diferenciável em (a, b) então é diferenciável em (x₁, x₂)
            e contínua [x₁, x₂]
            Pelo TVM
            ∃c ∈ (x₁, x₂) ⇒ f(x₂)-f(x₁) = f'(c)(x₂-x₁)

            Daí, pela hipótese do teorema temos que f'(c) = 0 e então f(x₂) = f(x₁)
            Como x₁, x₂ ∈ (a, b) são arbitrários, então todos os valores de f são iguais, ou seja
            f(x) = k = cte
