Conjuntos Numéricos

Conjunto dos números Naturais ℕ.

            ℕ = {0, 1, 2, 3, ...}
O conjunto dos números naturais provavelmente cresceu associado a processos de contagem.

Conjunto dos números Inteiros ℤ.

            ℤ = {..., -3, -2, -1, 0, 1, 2, 3, ...}
Considerar 0 e os negativos como "números" exigiu muita abstração dos homens.

Conjunto dos números Racionais ℚ.

            ℚ = {p/q, com p ∈ ℤ e q ∈ ℤ, q ≠ 0}
São racionais: 5, 0, -2, 1/3, 12/3 = 4, 15/10 = 3/2

Conjunto dos números Irracionais ℚ.

São todos numeros reais que não podem ser escritos na forma p/q com p e q inteiros, q ≠ 0. São iracionais: π, e (≅ 2, 718281 ...), √2, √3, ...

Os números reais, que é o conjunto básico de toda Matemática do ensino Fundamenta.
Vale 

            ℝ = ℚ∪I e ℚ∩I = ∅

Todo número real, racional ou irracional, pode ser aproximado por uma sequência de números racionais. A sequência: 3; 3,1; 3,14; 3,141; 3,1415; 3,14159; ... É formada por números racionais e se aproxima de π.
