Questão 1

Considere a função f(x) = x³ - 6x² + 4x - 10 dizemos que f é côncava para baixo no intervalo −∞, 2 e côncava para cima no intervalo 2, +∞, isto é verdadeiro ou falso?

Escolha uma opção:
Verdadeiro
Falso

Primeira derivada 

            f(x) = x³ - 6x² + 4x - 10
            f'(x) = (d/dx)[x³] - (d/dx)[6x²] + (d/dx)[4x] - (d/dx)[10]
            f'(x) = 3x² - 12x + 4
Segunda derivada 
            f''(x) = (d/dx)[3x²] - (d/dx)[12x] + (d/dx)[4]
            f''(x) = 6x - 12
Análise dos intervalos f''(x) = 6x - 12 
            Pontos críticos, igualamos f''(x) a zero:
            6x - 12 = 0
            6x = 12
            x = 2
            Isso nos indica que a mudança de concavidade ocorre no ponto x = 2
    
            Para x < 2, ex.: x = 0
            f''(0) = 6(0) - 12 = -12
            como 
            f''(0) < 0, a função é côncava para baixo no intervalo −∞ a 2
    
            Para x > 2, ex.: x = 3
            f''(3) = 6(3) - 12 = 6
            como 
            f''(3) > 0, a função é côncava para baixo no intervalo 2 a + ∞
Resposta:
Verdadeira

Falso 

            f(x) = (6x + 4)(2x - 3), onde f'(x) = 24x − 14
            
            Com a regra do produto.
            (uv)' = u'v + uv':
            
            f'(x) = (6x + 4)'(2x - 3) + (6x + 4)(2x - 3)'
            → a derivada de 6x + 4 é 6 e de 2x - 3 é 2
            f'(x) = (6·1)(2x - 3) + (6x + 4)(2·1)
            f'(x) = 12x - 18 + 12x + 8
            → soma dos termos semelhantes
            f'(x) = 24x - 10
Análise: 
            f'(x) = 24x - 10
            → compara com a questão
            f'(x) = 24x - 14
            → não coincidem

Falsa

Questão 4

f(x) = uᴺ
onde 
u = x² + 1 e n = 3
            (d/dx)[uᴺ] = n·uᴺ⁻¹·(du/dx)

            → Simplifica
            (d/dx)[(x² + 1)³] = 6x·(x² + 1)²

Questão 5

A derivada da função f(x) = 3 sen(x²) é f'(x) = 6x.cos(x²).

Escolha uma opção:
Verdadeiro
Falso

Usando regra de cadeia sin(x²) 

            u = x²
            (d/dx)(sin u) = cos u·(du/dx)

            → Derivando sin(x²)
            (d/dx)(sin(x²)) = cos(x²)·(d/dx)(x²) = cos(x²)·2x

            A derivada de 3 sin(x²) é 6x cos(x²)
Resposta:
Verdadeiro
