Cálculo de Limites

Teorema do Confronto (Sanduíche) Se: 

f(x) <= g(x) <= h(x)
em um intervalo aberto: 
I ∈ ℝ
e: 
x₀ ∈ I
e se: 
            lim f(x) = lim h(x) = L
            x ⇒ x₀     x ⇒ x₀
então também: 
            lim g(x) = L
            x ⇒ x₀

Limites Fundamental 

            lim (sin x)/x = 1   → A prova usa o Teorema do Confronto
            x ⇒ 0
Do circulo trigonométrico podemos ver que: 
sin x < x < tan x
e dividido por 
sin x
temos: 
            1 < (x/sin x) < 1/(cos x)
Como: 
            lim (1/cons x) = 1
            x ⇒ 0
Temos: 
            lim (x/sin x) = 1
e daí 
            lim (sin x/ x) = 1
            x ⇒ 0

O teorema do confronto com limites no infinito
Exemplo 1: 

            lim (sin x)/x = 0
            x ⇒ +∞
Basta observar que: 
0 < |(sin x)/x| = (|sin x|)/|x| < 1/|x|
Como: 
            lim  1/|x| = 0      lim  |sin x/x| = 0
            x ⇒ +∞              x ⇒ +∞
e daí: 
            lim  sin x/x = 0
            x ⇒ +∞
Pois a Função Módulo é contínua!

Multiplicação de termo conveniente 

            lim (cos x -1)/x = 0    Indeterminação do tipo 0/0
            x ⇒ 0
Multiplicamos o numerador e denominador por 
(cos x + 1)
Com um pouco de algebra trigonométrica obtemos: 
            lim (cos x -1)/x = lim ((sin x)/x) · lim (-sin x -1)/(cos x +1) = 1·0 = 0
            x ⇒ 0              x ⇒ 0             x ⇒ 0

Limite para Funções Limitadas
Dadas 

f(x) e g(x)
Definidas em um intervalo: 
I ⊆ ℝ
Se: 
            lim f(x) = 0  e  |g(x)| < M
            x ⇒ a
então: 
            lim [f(x)·g(x)] = 0
            x ⇒ a
pois: 
0 < |f(x)·g(x)| = |f(x)||g(x)| < |f(x)| M
e: 
            lim |f(x)] = 0
            x ⇒ a

Limite com Funções Contínuas
Sejam: 

f : I ⊆ ℝ e a ∈ I
A função 
f
é contínua no ponto 
a
se: 
            lim f(x) = lim f(x) = f(a)
            x ⇒ a⁺     x ⇒ a⁻
ou seja: 
            lim f(x) = L = f(a)
            x ⇒ a

Quais funções são contínuas e em quais pontos?
1. Todas as funções polinomiais em ℝ 2. Todas as funções trigonométricas e suas inversas em seu domínio de definição. 3. Todas as funções potências em ℝ 4. Todos os logaritmos em qualquer base em (0,∞) 5. Outras: módulo, raízes e-nesimas no seu domínio, etc.
