Teorema do Valor Intermediário
No experimento do vazamento lateral do líquido. O recipiente estava cheio e ficou vazio; Entre o início e o final, havia uma quantidade intermediária de liquido no recipiente. 

            _____________
            |           |
            |           |
            |    ___    |
            |   |   |   |
            |___|___|___|
Porque o volume de líquido, nesse contexto, é uma função contínua do tempo.

Teorema do Valor Intermediário I
Considere uma função contínua 

f : [a, b] ⇒ ℝ
em que o domínio 
[a, b]
é um intervalo fechado dos reais com 
f(a) < f(b)
Então, dado um número qualquer 
u ∈ (f(a)·f(b))
existe um ponto 
c ∈ (a, b)
tal que 
u = f(c)

Observações:
- No intervalo fechado 

[a, b] ⊂ ℝ
assumimos que 
a < b
- Se 
f(a) = f(b)
o teorema não se aplica. - Como 
f(a) e f(b) ∈ ℝ e f(a) ≠ f(b)
então um vai ser menor do que o outro.

Escolhemos, sem perda de generalidade, 
f(a) < f(b)

Teorema do Valor Intermediário II
Considere uma função contínua 

f : [a, b] ⇒ ℝ
em que o domínio 
[a, b]
é um intervalo fechado com 
f(a) < f(b)
Então, dado um número qualquer 
u tal que f(a) <= u <= f(b)
existe um ponto 
c em que a < c < b tal que u = f(c)
tal que 
u = f(c)

Demonstração. 

f(a) < u < f(b)
Seja 
S = {x ∈ [a, b] | f(x) <= u}
É claro que 
a ∈ S e S é ...
limitado por 
b
Então 
S ≠ ∅
é um conjunto não vazio de números reais com um limite superior e como tal, existe o extremo superior desses números, digamos 
c = sup S
isto é, 
c
é o menor número que ainda o mantém maior do que todos os demais de 
S

Teorema do Valor Intermediário III
Considere uma função contínua 

f : [a, b] ⇒ ℝ
em que o domínio 
[a, b]
é um intervalo fechado com 
f(a) < f(b)
Então, dado um número qualquer 
u tal que f(a) <= u <= f(b)
existe um ponto 
c em que a < c < b tal que u = f(c)

Demonstração. 

f(a) < u < f(b);
S = {x ∈ [a, b]|f(x) <= u};
c = sup S
Vamos provar que 
f(c) = u
Como 
f
é contínua 
∀ϵ > 0 |f(x) - f(c)| < ϵ então ∃δ > 0
em que 
|x - c| < δ
Assim 
∃x ∈ S com x ∈ (c - δ, c) |
f(c) < f(x) + ϵ <= u + ϵ
pela construção de 
S
Mas 
x ∈ (c, c + δ) ⇒ x ∉ S e f(c) > f(x) - ϵ > u - ϵ
Isto é: 
u - ϵ < f(c) < u + ϵ  ∀ϵ > 0 ⇒ f(c) = u

Teorema do Ponto Fixo
Se a função é sobrejetora então vai existir um ponto fixo 

f(x) x
Em outras palavras:
Se 
f : [a, b] ⇒ [a, b]
é contínua então existe 
c ∈ [a, b] | f(c) = c
Exemplo 
f : ℝ₊ ⇒ ℝ₊ com x ⇒  f(x) = 1 + 1/x
Alias, o ponto fixo nesse caso é único e pode ser encontrado a partir da equação 
f(x) = x
: 
            x = 1 + 1/x, x > 0
            x² - x - 1 = 0
            x = (1 + √5)/2
que é o número de ouro.

Teorema do Máximo e Mínimo I
Se 

f : [a, b] ⇒ ℝ
é contínua, então existe um ponto de máximo: 
c ∈ [a, b] tal que f(x) <= f(c)
e existe um ponto de mínimo: 
d ∈ [a, b] tal que f(d) <= f(x), ∀x ∈ [a, b]

Observação:
- O domínio de 

f
é um intervalo fechado em 
ℝ
- Existe pelo menos um ponto de cada estremo. 
f(d) <= f(x) <= f(c), ∀x ∈ [a, b]

Teorema do Máximo e Mínimo II
Se 

f : [a, b] ⇒ ℝ
é contínua, então existe um ponto de máximo: 
c ∈ [a, b]
e existe um ponto de mínimo: 
d ∈ [a, b] tais que f(d) <= f(x) <= f(c), ∀x ∈ [a, b]

Observação:
- Esses pontos extremos (mínimo ou máximo) são os absolutos ou globais.
- Algumas funções descontínuas podem ter esses extremos.
- O teorema não diz como encontrar esses extremos em geral.
- O modo gráfico é poderoso, mas pode enganar.
