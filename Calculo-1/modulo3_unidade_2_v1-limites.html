Cálculo de Limites

Limites Infinitos 

            lim f(x) = +∞
            x ⇒ x⁺₀
        
            lim f(x) = -∞
            x ⇒ x⁺₀
        
            lim f(x) = +∞
            x ⇒ x⁻₀
        
            lim f(x) = -∞
            x ⇒ x⁻₀
A reta 
x = x₀
é assíntota

Limites no Infinito 

            lim f(x) = L
            x ⇒ ±∞
        
            lim f(x) = ±∞
            x ⇒ ±∞
A reta 
y = L
é assíntota horizontal

Propriedade Básicas de Limites 

            lim[f(x) ± g(x)] = lim f(x) ± lim g(x)
            x ⇒ a              x ⇒ a      x ⇒ a
        
            lim[f(x)·g(x)] = lim f(x)·lim g(x)
            x ⇒ a            x ⇒ a    x ⇒ a
        
            lim f(x)/g(x)] = lim f(x)/lim g(x)    desde que lim g(x) ≠ 0
            x ⇒ a            x ⇒ a    x ⇒ a       x ⇒ a
se 
            lim f(x) = ±∞   então   lim 1/f(x) = 0
            x ⇒ a                   x ⇒ a

Exemplos 

            lim(3x² + 1) / (x³ + 4) = lim (3x² + 1) / lim(x³ + 4) = 4/5
            x ⇒ 1                     x ⇒ 1           x ⇒ 1
        
            lim P(x)/P(a)       para qualquer polinômio     P(x)
            x ⇒ a
        
            lim P(x) / Q(x) = lim P(x) / lim Q(x) = P(a)/Q(a), se Q(a) ≠ 0
            x ⇒ a             x ⇒ a      x ⇒ a

Formas Indeterminadas
Muitos limites interessantes caem nos seguintes casos que são indeterminados à primeira vista.
Ex.: 

            lim x²-1/x²+1 = lim x²-1/x²+1 = +∞/+∞
            x ⇒ +∞          x ⇒ +∞
Outras formas indeterminadas 
            0/0, ∞/∞, ∞-∞, 0°, 1^∞, ∞° e 0.∞ e ∞.0

Limites no Infinito de Funções Racionais
Exemplos: 

            lim x²-1/x²+1 = lim (1 - 1/x²) / lim (1 + 1/x²) = 1/1 = 1
            x ⇒ +∞          x ⇒ +∞           x ⇒ +∞
        
            lim x-1/x²+1 = lim (1/x - 1/x²) / lim (1 + 1/x²) = 0
            x ⇒ -∞         x ⇒ -∞             x ⇒ -∞
        
            lim (x³ - 1)/(x² + 1) = lim (x - 1/x²) / lim (1 + 1/x²) = -∞
            x ⇒ -∞                  x ⇒ -∞           x ⇒ -∞

Limites no Infinito de Funções Racionais (Geral)
Dados dois polinômios 

P(x) e Q(x)
            grau(P) < grau(Q) ⇒ lim P(x)/Q(x) = 0
                                x ⇒ ±∞ 
        
            grau(P) > grau(Q) ⇒ lim P(x)/Q(x) = ±∞
                                x ⇒ ±∞ 
            Obs: Neste caso o sinal deve ser observado cuidadosamente.
