Inequações Produto e Inequações Quociente

Inequações Produto e Quociente

        P(x) > 0

        P(x) > 0 / Q(x) > 0

        Com P(x) e Q(x) polinomiais

Exemplo

        x⁵ - 5x⁴ + 7x³ - 5x² + 6x / -x -3 <= 0

Preliminares:

1) Todo polinômio P(x) pode ser fatorado como produto de polinômio de grau <= 2

2) Se α é a raiz de um polinômio P(x), isto é, P(α) = 0, então P(x) pode ser fatorado como:
P(x) = (x - α)· Q(x) 

        O polinômio P(x) = x⁵ - 5x⁴ + 7x³ - 5x² + 6x
        Pode ser fatorado como
        P(x) = x·(x² + 1)·(x² - 5x + 6)
        
        P(x) = x·(x² + 1)·(x - 2)(x - 3)

        Cada termo da composição é chamado de "Irredutível"

        Irredutíveis: x, x²+1, x-2, x-3

Resolver a inequação

        x⁵ - 5x⁴ + 7x³ - 5x² + 6x > 0
        x·(x² + 1)·(x - 2)(x - 3) > 0

        x           -- | +
                       0
        x² + 1      +     +
        x - 2           -- | +
                           2
        x - 3               -- | +
                               3
    
        | Intervalo | x   | x² + 1 | x - 2 | x - 3 |
        |-----------|-----|--------|-------|-------|
        | x < 0     | -   | +      | -     | -     |
        | 0 < x < 2 | +   | +      | -     | -     |
        | 2 < x < 3 | +   | +      | +     | -     |
        | x > 3     | +   | +      | +     | +     |
    
        S = {x ∈ ℝ | 0 < x < 2 ou 3 < x}
        S = ]0,2[∪]3, ∞[

Resolver a inequação quociente

        x⁵ - 5x⁴ + 7x³ - 5x² + 6x / x - 3 <= 0
        
        Condições de existência (Domínio):
        -x -3 ≠ 0 ⇔ x ≠ -3
        ℝ \{-3}

Estudo de Sinal de -x - 3:

        Intervalos  | -x - 3
        ------------|-------
        x < -3      | Positivo (+)
        -3 < x      | Negativo (-)
        x > -3      | Negativo (-)

        Ao considerar o estudo de sinal do denominador −x−3 e a continuidade dos termos do numerador, observamos que a expressão original será negativa quando x>−3.
        Contudo, devemos lembrar que x=3 é um ponto de descontinuidade devido à divisão por zero.

        Portanto, a solução para a inequação é
        x ∈ (−∞,−3)∪(3,+∞), excluindo o ponto de descontinuidade
        x=3.
