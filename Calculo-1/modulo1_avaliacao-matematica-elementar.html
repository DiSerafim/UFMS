Questão 1

Encontre o conjunto de soluções da inequação (x - 1)×(2 - x) ≥ 0

a. S = {x ∈ R/ x ≥ 2}
b. S = {x ∈ R/ X ≤ 1 e x ≥ 2}
c. S = {x ∈ R/ 1 ≤ x ≤ 2}
d. S = {x ∈ R/ x ≥ 1 e x ≤ 2}
e. S = {x ∈ R/ x ≤ 1} 

        x - 1 = 0 e 2 - x = 0                       - ponto crítico.
        (x - 1) = 0 ⇒ x = 1
        (2 - x) = 0 ⇒ x = 2
        x = 1 e x = 2
        (-∞, 1), (1, 2) e (2, ∞)                    - intervalos (x - 1)·(2 - x)
        (0 - 1)·(2 - 0) = -1 · 2 = -2 < 0           - intervalo (-∞, 1), x = 0
        (1.5 - 1)·(2 - 1.5) = 0,5 · 0,5 = 0.25 > 0  - intervalo (1, 2), x = 0
        (3 - 1)·(2 - 3) = 2 · -1 = -2 < 0           - intervalo (2, ∞), x = 3

        intervalo (-∞, 1), x = 0 → Negativo
        intervalo (1, 2), x = 0  → Positivo
        intervalo (2, ∞), x = 3  → Negativo

        Resposta:
        c. S = {x ∈ R/ 1 ≤ x ≤ 2}

Questão 4

Questão 6

Seja dada a inequação (x - 1)(x - 4) ≤ 0.
Considerando o produto do primeiro número inteiro da solução e o último número inteiro da solução que a satisfazem.
É correto concluir que:

a. O produto pedido é 4.
b. O maior deles é múltiplo de 3.
c. A soma de todos eles é nove.
d. O produto pedido de todos é um número negativo.
e. O produto pedido é seis. 

        (x - 1)(x - 4) ≤ 0
        x - 1 = 0 e x - 4 = 0               - ponto critico
        x = 1 e x = 4
        (-∞, 1), (1, 4) e (4, ∞)            - intervalos
        (0 - 1)·(0 - 4) = -1 · -4 = 4 > 0   - intervalos (-∞, 1), x = 0
        (3 - 1)·(3 - 4) = 2 · -1 = -2 < 0   - intervalos (1, 4), x = 3
        (5 - 1)·(5 - 4) = 4 · 1 = 4 > 0     - intervalos (4, ∞), x = 5

        (-∞, 1) → Positivo
        (1, 4)  → Negativo
        (4, ∞)  → Positivo

        Portanto, os números inteiros que satisfazem a inequação são 1, 2, 3 e 4.

        O produto do primeiro número inteiro da solução (1) pelo último número inteiro da solução (4) é 1×4=4.

        Resposta:
        a. O produto pedido é 4.

Questão 9

Encontre o conjunto de soluções da inequação x² – 6x + 8 < 0

a. S = {x ∈ R / 2 < x < 4}
b. S = {x ∈ R / x < 4}
c. S = {x ∈ R / x < 2}
d. S = {x ∈ R / 2 ≥ x ≥ 4}
e. S = {x ∈ R / 2 ≤ x < 4} 

        x² – 6x + 8 < 0
        x² – 6x + 8 = (x - 4)·(x - 2) < 0       - fatora
        x - 4 = 0 ⇒ x = 4                       - ponto critico
        x - 2 = 0 ⇒ x = 2                       - ponto critico
        f(0) = (0 - 4)·(0 - 2) = 4·2 = 8 > 0    - ponto critico (-∞, 2), x = 0
        f(3) = (3 - 4)·(3 - 2) = -1·1 = -1 < 0  - ponto critico (2, 4), x = 3
        f(5) = (5 - 4)·(5 - 2) = 1·3 = 3 > 0    - ponto critico (4, ∞), x = 5

        (-∞, 2) → Positivo
        (2, 4)  → Negativo
        (4, ∞)  → Positivo

        Resposta:
        a. S = {x ∈ R / 2 < x < 4}
