Tabela Geral de Derivada

Na tabela que se segue u, v são funções deriváveis de x e c, α, a são constantes reais. 

            1. y = c ⇒ y' = 0
               y = 5 ⇒ y' = 0
               (5)' = 0
               y = f(x) ⇒ y' = f'(x)

            2. y = x ⇒ y' = 1

            3. y = c·u ⇒ y' = c·u
               y = 3·x ⇒ y' = 3(x)' = 3·1 = 3
               y' = 3

            4. y = u ± v ⇒ y' = u ± v
               y = u-v = u+(-v)
               y' = u'+(-v)' = u' -v
            
            5. y = u·v ⇒ y' = u'·v + u·v'
               y = (3x + 4)·(5x + 6)
               y' = (3x + 4)'·(5x + 6)+(3x + 4)·(5x + 6)'
               y = 3·(5x + 6) + (3x + 4)·5
               y = 30x + 38
            
            6. y = u/v ⇒ y' = (u'·v - u·v')/v²
               y = 2x²/x²-1
               y' = (4x(x²-1)-2x²·2x)/(x²-1)²
               y' = (4x³ - 4x - 4x³)/(x²-1)²
               y = -4x/(x²-1)²

            7. y = uᴬ ⇒ y' = auᴬ⁻¹·u'·(a ≠ 0)
               y = (x³ + 3x)⁵
               y'= 5(x³ + 3x)⁴·(x³ + 3x)'
               y = 5(x³ + 3x)⁴·(3x² + 3)
               y = 15(x² + 1)·(x³ + 3x)⁴

            8. y = aᵁ ⇒ y' = aᵁ·la·u, (a > 0, a ≠ 1)
               y = 5x³
               y' = 5x³·ln·5·(3x²)
               y = 3x²·(ln·5)·5x²

                y = log₅ x3
                y' = 1/(x³·ln 5)·3x²
