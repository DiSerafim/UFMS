Funções 1: Noções básicas

É um trio composto por um conjunto de saída (domínio X), um conjunto de chegada (contra-domínio Y) e um tipo de relação entre esses conjuntos. 

            f:X → Y
            x → y = s(x)
Cada (e todo) x em X tem um único correspondente y em Y 

        X                 Y
        +---+            +---+
        |   |            |   |
        | x | ---------->| y |
        |   |            |   |
        +---+            +---+
        
        X representa o conjunto de saída.
        Y representa o conjunto de chegada.
        x é um elemento em X.
        y é o correspondente de x em Y, ou seja, y=f(x).

Exemplo:
Uma partícula em queda livre, sem resistência tem a sua altura 'h' como uma função do tempo 't'. 

            h:T → Y
            t → y = h(t) = 3 - 5t²
            t >= 0

            (t, 3 - 5t²), t >= 0

Exemplo:
Uma elipse completa não é o gráfico de uma função.
Um pedaço (arco) apropriado da elipse pode ser uma função.
Tenha claro quais são os conjuntos:
- Domínio.
- Contra-domínio.
- Imagem de uma função. 

             1 |          ***
               |        *       *
               |      *           *
               |     *             *
               |    *               *
               |    *               *
               |     *             *
             0 |      *           *
               |        *       *
               |           ***
            -1 +-----------------------
            -2    -1     0     1     2

            Suponha a elipse com o eixo horizontal (eixo x) variando de -2 a 2 e o eixo vertical (eixo y) variando de -1 a 1. Vamos considerar o quarto superior direito da elipse.

            Neste caso:

            - O domínio seria o intervalo de valores ao longo do eixo horizontal (x) que cobre a seção da elipse que estamos considerando.
            - O contra-domínio seria o intervalo de valores ao longo do eixo vertical (y).
            - O conjunto imagem seria a seção da elipse que corresponde a todos os valores possíveis de y para os valores de x no domínio especificado.

            Embora essa representação gráfica mostre uma seção da elipse, ela ainda não é o gráfico de uma função, pois para cada valor de x nessa seção, ainda existem múltiplos valores possíveis de y, o que viola a definição de função, onde cada valor de x deve estar associado a no máximo um valor de y.
