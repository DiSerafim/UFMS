Construção e definição de novas funções

Função Módulo (valor absoluto) 

                  ┌   
                  | x  se  x >= 0
            |x| = <    
                  | -x  se  x < 0
                  └
Também pode ser escrita como: 
            |x| = √x²    Para todo x real
Obs: Esta forma é muito útil para facilitar cálculos envolvendo módulo. 

            |
            |       /|
            |      / |
            |     /  |   x
            |    /   |
            |   /    |
            |  /     |  x
            | /      |
            |/_______|__________
                    -1     0     1
Neste gráfico:

- No eixo horizontal (x), temos os valores reais.
- No eixo vertical, temos os valores correspondentes de ∣x∣ (o valor absoluto de x).
- A função ∣x∣ retorna x se x for maior ou igual a zero e retorna −x se x for menor que zero.
- Isso cria a forma de "V" que representa a função módulo, onde a linha cruza a origem (0,0) e reflete o valor negativo para o seu correspondente positivo.

Função Exponencial 

            f(x) = aˣ onde a > 0 e a ≠ 1
            se a > 1, a função é crescente
            se a < 1, a função é decrescente
            Dom(f) = ℝ e Im(f) = (0, + ∞)
O número a é chamado de base da função exponencial.

Função Logarítmica
Definida como função inversa da Função Exponencial 

            Se y = f(x) = aˣ então x = f⁻¹(y) = logₐy
Nas mesmas variáveis do sistema cartesiano da função original podemos escrever: 
            y = g(x) = logₐx
        
            Dom(g) = (0, + ∞) e Im(g) = ℝ
Se a = e obtemos a função Logarítmica Neperiana e denota-se como y = ln x

vamos considerar a função logarítmica na base a como y=logₐx, onde a é a base do logaritmo.

Vamos representar o gráfico da função logarítmica na base a em um sistema cartesiano: 

            ^
            |  *  
            |    *  
            |      *
            |        *  
            |         *
            |           *
            |             *
            |               *
            |                 *
            |                   *
            |                     *
            |                       *
            |                         *
            |                           *
            +--------------------------------->
                0   1   2   3   4   5   6   7 ...
Neste gráfico:

- No eixo horizontal, temos valores positivos de x.
- No eixo vertical, temos os valores correspondentes de logₐx.
- O gráfico da função logarítmica mostra como os valores crescem de forma mais lenta à medida que x aumenta.

Funções Trigonométricas (periódicas) 

            y = f(x) = sen(x)
            Dom(f) = ℝ
            Im(f) = [-1, 1]
                        
            y = g(x) = cos(x)
            Dom(g) = ℝ
            Im(g) = [-1, 1]

funções trigonométricas y=sin(x) e y=cos(x), que são funções periódicas.

Funções Tangente e Cotangente 

            y = f(x) = tan(x)       y = f(x) = cot(x)
            Dom(f) = ℝ - {xₙ = (2n + 1)π/2, n ∈ ℤ}
            Im(f) = ℝ   Para ambas as funções.

gráfico que representa as funções y=tan(x) e y=cot(x), que são funções trigonométricas. 

            ^
            | ***********       ***********
            |           **   ***
            |             **
            |              *            ***********
            |               **        **
            |                 **    **
            |                   ****
            |                    * 
            |                      *        ***********
            |                       **    **
            |                         ** **
            |                           **

            -π/2      -π       0        π       π/2
Neste gráfico:

- No eixo horizontal, estão os valores de x em radianos.
- No eixo vertical, estão os valores correspondentes de y para as funções y=tan(x) e y=cot(x).
- A função y=tan(x) (tangente) tem assíntotas verticais em x=(2n+1) π/2 para n pertencente aos números inteiros (ℤ).
- A função y=cot(x) (cotangente) tem comportamento semelhante com assíntotas verticais em x=nπ para n pertencente aos números inteiros (ℤ).

Esses gráficos representam o comportamento das funções tangente e cotangente ao longo do intervalo de −π/2 a π/2 em radianos.

Funções Secante e Cossecante 

            y = f(x) = sec(x)                       y = f(x) = cot(x)
            Dom(f) = ℝ - {xₙ = 2nπ/2, n ∈ ℤ}        Dom(f) = ℝ - {xₙ = 2πn/2, n ∈ ℤ}
            Im(f) = (-∞, 1]∪[1, ∞)                  Im(f) = (-∞, 1]∪[1, ∞)

Classificação de funções
- Injetoras, sobrejetoras, bijetoras.
- Crescente ou decrescente.
- Contínuas ou descontinuas (por partes) - com saltos finito ou não.
- Limitadas ou não limitadas.
- Diferenciável ou não.
- Integrável ou não.
