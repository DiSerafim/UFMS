Questão 3

Decida se o resultado do limite é verdadeiro ou falso: 
            lim (6x⁴ + x² + 2)/−5x⁶ − 3 = −∞
            x→−∞

Escolha uma opção:
Verdadeiro
Falso 

        lim (6x⁴ + x² + 2)/-5x⁶ - 3 = -∞
        x→-∞

        Dividimos o numerador e o denominador por x⁴
        lim (6x⁴ + x² + 2)/x⁴ / (-5x⁶ - 3)/x⁴
        x→-∞

        Resulta em
        lim (6 + 1/x² + 2/x⁴) / -5 - (3/x⁴)
        x→-∞

        Simplifica
        lim (6 + 1/x² + 2/x⁴) / -5 - (3/x⁴) = (6 + 0 + 0) / -5 -0 = 6/-5 = -(6/5)
        x→-∞

        Portanto, o resultado do limite não é -∞ é - -(6/5)

        Resposta: 

        Falsa.

Questão 4

O resultado do limite 
            lim 1/√x−√3 = ?
            x→3+
é?

a. não existe
b. −∞
c. 23–√
d. +∞
e. 1/23–√ 

        lim 1/√x−√3 = ?
        x→3+

        3+, estamos nos aproximando de 3 pelo lado direito, ou seja, valores um pouco maiores que 3. 

        Portanto, podemos reescrever a expressão para lidar com a raiz no denominador:

        lim 1/√x−√3 x (√x+√3)/(√x+√3)
        x→3+

        Resulta em:
        lim (√x+√3)/x-3
        x→3+

        Substituição direta de x=3
        lim (√x+√3)/x-3 = (√3+√3)/3-3 = 2√3/0+ = +∞
        x→3+

        Resposta:
        d. +∞

Questão 10

Decida se o resultado do limite é verdadeiro ou falso: 
            lim (6x³ + x² + 2)/3x³ − 3 = 2
            x→−∞

Escolha uma opção:
Verdadeiro
Falso 

        lim (6x³ + x² + 2)/3x³ − 3
        x→−∞

        Simplifica
        lim (6+(1/x) + (2/x³)) / 3 - (3/x³)
        x→−∞

        Quando x tende a −∞, todos os termos que possuem 1/x ou 1/x³ vão para zero:
        (6 + 0 + 0)/3 - 0 = 6/3 = 2

        Resposta:
        Verdadeiro.
