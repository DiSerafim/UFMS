Derivadas e Taxas de Crescimento de Funções
Teorema: Seja 

f : I → ℝ
diferenciável 
            a) f'(x) > 0 em I → f(x) é crescente em I

            b) f'(x) < 0 em I → f(x) é decrescente em I
Demonstração de (a): Usa o TVM.
Sejam dois pontos 
x₁ < x₂ em I → x₂ - x₁ > 0
Pelo TVM 
∃c ∈ (x₁, x₂)
tal que 
f(x₂)-f(x₁) = f'(c)(x₂-x₁)
f'(x) > 0, ∀x ∈ I → f'(c) > 0
O que implica 
f(x₁) < f(x2) → f é crescente
obs: Demonstração de (b) similar é similar

Pontos Críticos de uma função
Def 1: Seja

            f'(c) = 0 ou ∄f'(c)
Def 2: Um ponto 
c ∈ I

            V = (c - e, c + e)

            f(x) <= f(c), ∀x ∈ V
Um ponto 
c ∈ I
é um mínimo local se existe uma vizinhança: 
            V = (c - e, c + e)
            tal que
            f(x) >= f(c), ∀x ∈ V
