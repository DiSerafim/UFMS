Variável composta heterogênea

Variável simples heterogêneas.

Capaz de armazenar um única informação.
Podem ser de dois tipos:
- Homogênea.
- Heterogênea.

- Capaz de armazenar uma única informação, que pode ser dividida em "pedaços" de diferentes tipos.
- Antes de declarar uma variável simples heterogênea, necessário definir quais serão os "pedaços" dessa variável.

Criando um tipo Heterogêneo

Em Python, para criarmos uma variável simples heterogênea precisamos criar uma classe.
Classe:
- Define quais os "pedaços" da sua variável heterogênea.
Exemplo:
- Uma variável chamada n que precisa armazenar o nome, a nota 1 e a nota 2 de um aluno.
- Criaremos uma classe para definir o que a variável n deve conter. 

            >>> class Nota:
            ...     def __init__(self, nome, nota1, nota2):
            ...         self.nome = nome
            ...         self.nota1 = nota1
            ...         self.nota2 = nota2
Nota - Nome da classe. Deve sempre aparecer após a palavra chave 'class'.
def __init__ - Define um método(equivalente a uma função) que dirá quais os "pedaços" da variável nota.
self - Deve sempre ser o primeiro parâmetro de qualquer método.
nome, nota1, nota2 - Os nomes dos "pedaços" da sua variável, separados por vírgula. - Esses pedaços são chamados de atributos.

Essas três linhas "declaram" os atributos de uma Nota.

Declarando e acessando uma Variável Heterogênea

            self.nota1 = nota1
            ...         self.nota2 = nota2
            ... 
            >>> n = Nota("Jamily", 10.0, 5.5)
            >>> n2 = Nota("Rosa Maria", 6.0, 7.0)
            >>> print(n.nome)
            Jamily
            >>> print(n2.nota1)
            6.0

Alterar Nome 

            >>> class Nota:
            ...     def __init__(self, nome, nota1, nota2):
            ...         self.nota2 = nota2
            ...         self.nome = nome
            ...         self.nota1 = nota1
            ...     def alteraNome(self, nome):
            ...         self.nome = nome
            ... 
            >>> n = Nota("Wallace", 10.0, 5.5)
            >>> n.alteraNome("Wallace2")
            >>> print(n.nome)
            Wallace2

Ponteiro

O que acontece quando declaramos uma variável de um tipo heterogêneo.
Exemplo:
n = Nota("Ziguifrido", 3.0, 4.5)
O que a variável n armazena?

Toda variável heterogênea armazena o endereço dos seus atributos na memória.
Uma variável que armazena um endereço é chamada de ponteiro. Exemplo:
n = Nota("Ziguifrido", 3.0, 4.5) 

            Variável n ---> [Instância de Nota]
                            |   Nome: "Ziguifrido"
                            |   Nota 1: 3.0
                            |   Nota 2: 4.5

pythontutor

Variável composta heterogênea

Variável capaz de armazenar um conjunto de informações.
Exemplo:
Vetores de inteiros. 

            >>> v = []
            >>> v.append(10)
            >>> v.append(15)
            >>> v.append(8)
            >>> print(v)
            [10, 15, 8]
Vetores de notas. 
            >>> v = []
            >>> v.append(Nota("Samuel", 6, 7))
            >>> v.append(Nota("Wallace", 7, 8))
            >>> print(v)

Resumo

Variáveis compostas heterogêneas são conjuntos de variáveis heterogêneas.
Uma variável composta (vetor) também é um ponteiro.
Uma variável composta heterogênea armazena um conjunto de ponteiros.
