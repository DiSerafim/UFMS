Análise assintótica: ordens O, Ω e Θ

No pior caso, o tempo cresce na proporção do quadrado do tamanho da sequência.
Podemos formalizar estes conceitos com as notações O, Ω e Θ, que permitem fazer uma comparação assintótica de funções.

Notação Ω

Dizemos que T(n) é Ω(f(n)) se existem constantes positivas c e n₀ tais que:
T(n) ≥ c f(n) para todo n ≥ n₀. 

        Exemplo 1:
            Demonstrar que 6n+5 é Ω(n)
        Prova:
            6n+5 ≥ 6n para todo n ≥ 0 => c=6 e n₀=0
    
        Exemplo 2:
            Demonstrar que n²-2 é Ω(n²)
        Prova:
            n²-2 = n²/2+(n²/2-2) ≥ n²/2 se (n²/2-2) ≥ 0, isto é, para todo n ≥ 2 => c=1/2 e n₀=2.
    
        Exemplo 3:
            Demonstrar que nlgn é Ω(n)
        Prova:
            lg n ≥ 1 para todo n ≥ 2 e portanto nlg n ≥ n => c=1 e n₀=2

Notação Θ

Dizemos que T(n) é Θ(f(n)) se existem constantes positivas c₁ e c₂ e n₀ tais que:
c₁f(n) ≤ T(n) ≤ c₂f(n) para todo n ≥ n₀. 

        Exemplo 1:
            Demonstrar que n²-2 é Θ(n²)
        Prova:
            n²-2 ≥ n²/2+(n²/2-2) ≥ n²/2 se (n²/2-2) ≥ 0, isto é, para todo n ≥ 2 => c₁=¹/₂ e n₀₁=2

            n²-2 ≤ n² para todo n ≥ 0 => c₂=1 e n₀₂=0.

            n²/2 ≤ n²-2 ≤ n², c₁=¹/₂, c₂=1, n₀=max{n₀₁,n₀₂} = 2

Θ(1) - Constante: A função ou algoritmo tem tempo de execução constante, independentemente do tamanho.

Θ(log n) - Logarítmica: O tempo de execução cresce de forma logarítmica à medida que o tamanho da entrada aumenta.

Θ(n) - Linear: O tempo de execução aumenta linearmente com o tamanho da entrada.

Θ(n log n) - Linearítmica: Têm um tempo de execução proporcional ao tamanho da entrada multiplicado pelo logaritmo desse tamanho.

Θ(n²), Θ(n²), ... - Quadrática, Cúbica, e por diante: O tempo de execução aumenta quadrática, cúbica, ou em graus superiores conforme a entrada aumenta.

Θ(2ᴺ), Θ(n!), ... - Exponencial e Fatorial: Têm um tempo de execução que cresce exponencialmente ou de forma fatorial em relação ao tamanho da entrada.

Propriedades

Transitividade:

T(n) é Θ(f(n)) e f(n) é Θ(g(n)) então T(n) é Θ(g(n))
T(n) é O(f(n)) e f(n) é O(g(n)) então T(n) é O(g(n))
T(n) é Ω(f(n)) e f(n) é Ω(g(n)) então T(n) é Ω(g(n))

Reflexividade:

T(n) é Θ(T(n))
T(n) é O(T(n))
T(n) é Ω(T(n))

Simétrica:

T(n) é Θ(f(n)) então f(n) é Θ(T(n))

Antissimétrica:

T(n) é O(f(n)) então f(n) é Ω(T(n))
T(n) é Ω(f(n)) então f(n) é O(T(n))
