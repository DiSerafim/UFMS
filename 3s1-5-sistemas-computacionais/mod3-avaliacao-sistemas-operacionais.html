Pergunta 2

A unidade de gerenciamento de memória (MMU, do inglês Memory Management Unit) é um módulo do software de suporte ao mecanismo de memória virtual e faz o mapeamento dos endereços físicos em endereços virtuais.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Falso. ✅

Resolução:

A unidade de gerenciamento de memória (MMU, do inglês Memory Management Unit) é um componente de hardware, não de software, responsável pelo mapeamento dos endereços virtuais em endereços físicos.

Pergunta 4

Assinale a alternativa que descreve o que é um Sistema Operacional (SO).

  • a. O SO faz parte do hardware do computador e é através dele que os programas do usuário utilizam os recursos do hardware.
  • b. O SO atua apenas como uma interface para a comunicação entre o hardware e o usuário.
  • c. O SO é um sistema que tem a única função permitir a comunicação entre computadores.
  • d. O SO é um dispositivo conectado aos circuitos do processador do computador e tem como função gerenciar os recursos de hardware.
  • e. O SO é um software cuja função é gerenciar os recursos do hardware do computador e fornecer uma interface entre o computador e os programas do usuário.
Resposta: e. ✅

Resolução:

  • Um Sistema Operacional (SO) é um software essencial que atua como intermediário entre o hardware do computador e os programas do usuário. Ele gerencia os recursos do hardware, como a CPU, memória, dispositivos de entrada/saída e armazenamento, além de fornecer uma interface para que os programas possam utilizar esses recursos de maneira eficiente e segura.

Pergunta 5

Assinale a alternativa que NÃO representa uma função do Sistema Operacional.

  • a. Organizar os arquivos no disco.
  • b. Controlar as operações de entrada/saída.
  • c. Controlar a execução dos programas do usuário.
  • d. Controlar a alocação de memória para os programas do usuário.

Resposta: e. ✅

Resolução:

  • Compactar arquivos é uma função específica de programas utilitários que podem ser executados no sistema operacional, mas não é uma função fundamental ou obrigatória do sistema operacional em si.

Pergunta 7

Assinale os recursos que são compartilhados pelos threads de um processo multithreaded.

  • a. Área de dados na memória
  • b. Área de pilha na memória.
  • c. Valores dos registradores do processador.
  • d. Área de código na memória.
Resposta: a, d. ✅

Resolução:

  • a: Todos os threads de um processo compartilham a mesma área de dados. Isso inclui variáveis globais e estruturas de dados que são usadas pelo processo. O compartilhamento da área de dados permite que threads se comuniquem e compartilhem informações facilmente, mas também requer mecanismos de sincronização para evitar condições de corrida.
  • d: Os threads de um processo compartilham o mesmo código executável. Isso significa que eles executam as mesmas instruções, embora possam estar em diferentes pontos de execução ao mesmo tempo. O compartilhamento do código evita a duplicação e economiza memória.

Pergunta 8

O buffer para tradução de endereços (TLB, do inglês Translation Lookaside Buffer) é uma pequena memória em hardware usada para acelerar o mapeamento de endereços virtuais em endereços físicos e contém algumas informações da tabela de páginas do processo.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Verdadeiro. ✅

Resolução:

  • A Translation Lookaside Buffer (TLB) é uma pequena e rápida memória em hardware que é usada para armazenar uma cache das entradas da tabela de páginas. A TLB acelera o processo de mapeamento de endereços virtuais para endereços físicos, mantendo as traduções recentes em um local de acesso rápido. Isso reduz significativamente o tempo necessário para realizar a tradução de endereços, melhorando a eficiência do sistema de memória virtual.

Pergunta 9

Sobre o mecanismo de memória virtual, assinale a alternativa INCORRETA.

  • a. É implementado pelo Sistema Operacional com suporte do hardware.
  • b. É implementado juntamente com um mecanismo de alocação particionada, como paginação, segmentação ou segmentação paginada.
  • c. Tira proveito do princípio de localidade apresentado pela maioria dos programas.
  • d. Permite executar um processo cujo espaço de endereçamento seja maior que a memória principal (memória física).
  • e. Não permite manter vários processos na memória principal (memória física) ao mesmo tempo.
Resposta: e. ✅

Resolução:

  • O mecanismo de memória virtual permite que múltiplos processos sejam mantidos na memória principal ao mesmo tempo, cada um com seu próprio espaço de endereçamento virtual. Isso é uma das principais vantagens da memória virtual, pois aumenta a utilização eficiente da memória física e melhora a capacidade do sistema de multitarefas.

Pergunta 10

Sobre o mecanismo de memória virtual com paginação, assinale a alternativa INCORRETA.

  • a. As transferências entre a memória principal (memória física) e disco são sempre em páginas inteiras.
  • b. A memória principal (memória física) é vista como consistindo de partes de tamanho fixo denominados de molduras de página (frames).
  • c. As páginas e molduras de página (frames) têm o mesmo tamanho.
  • d. O espaço de endereço virtual de um processo é visto como consistindo de partes de tamanho fixo denominadas páginas.
  • e. As páginas de um processo que estão presentes na memória principal (memória física) precisam ocupar molduras de páginas contíguas.
Resposta: e. ✅

Resolução:

  • As páginas de um processo não precisam ocupar molduras de páginas contíguas na memória principal. Elas podem estar dispersas ao longo da memória física. Isso é uma das vantagens da paginação, pois permite uma alocação mais flexível da memória física.
