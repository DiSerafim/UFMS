Questionário ✅ ❌

Pergunta 1

Assinale as alternativas que representam informações providas nos campos de uma instrução de máquina.

  • a. Os operandos destino usado pela instrução.
  • b. O comentário da instrução.
  • c. O código da operação representada pela instrução.
  • d. O rótulo (do inglês, label) representa o endereço da instrução na memória.
  • e. Os operandos fonte usados pela instrução.
Resposta: a, c, e. ✅

Resolução:

  • a: Representa o local onde o resultado da operação será armazenado.
  • c: Indica qual operação (como ADD, SUB, MPY, DIV, etc.) deve ser executada.
  • e: Indicam os dados que a operação utilizará para realizar a instrução.

Pergunta 2

Usando instruções aritméticas genéricas de 3 endereços (com um operando destino e dois operandos fontes, nessa ordem), indique como a expressão A = B - (C * D) + E pode ser traduzida. Isto é, assinale a alternativa que mostra a sequência de instruções que implementa a expressão.

  • a. MUL A, C, D; SUB A, A, B; ADD A, A, E;
  • b. SUB A, B, C; MUL A, A, D; ADD A, A, E;
  • c. SUB A, B, C; ADD A, D, E; MUL A, A, A;
  • d. MUL A, C, D; SUB A, B, A; ADD A, A, E;
Resposta: d. ✅

Resolução:

                Primeiro, calcular C * D e armazenar o resultado em um temporário T1:
                MUL T1, C, D

                Depois, calcular B - T1 e armazenar o resultado em outro temporário T2:
                SUB T2, B, T1

                Finalmente, adicionar E ao resultado armazenado em T2 e armazenar o resultado final em A:
                ADD A, T2, E

                A sequência correta de instruções seria:
                MUL T1, C, D
                SUB T2, B, T1
                ADD A, T2, E

Pergunta 3

Instruções de máquina com 2 endereços em geral são mais curtas (em número de bits) do que instruções de máquina com 3 endereços.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Verdadeiro. ✅

Resolução:

  • Cada endereço (operando) ocupa um número fixo de bits no formato da instrução. Reduzir o número de operandos reduz a quantidade total de bits necessária para representar a instrução.
    • Instruções de 2 endereços: Têm um campo para o endereço do operando de destino e um campo para o endereço do operando fonte. Geralmente, uma das operações é realizada "in-place" (isto é, o resultado é armazenado em um dos operandos de entrada).
    • Instruções de 3 endereços: Têm campos separados para o endereço do operando de destino e os endereços de dois operandos fontes. Isso permite mais flexibilidade nas operações, pois o resultado pode ser armazenado em um local diferente dos operandos de entrada, mas exige mais bits para codificar a instrução.
  • Por exemplo, suponha que um endereço de operando exige 8 bits. Uma instrução de 2 endereços precisa de 16 bits para os operandos, enquanto uma instrução de 3 endereços precisa de 24 bits para os operandos.

Pergunta 4

O termo conjunto de instruções do processador (ou ISA, do inglês Instruction Set Architecture) refere-se a:

  • a. O conjunto de registradores internos do processador.
  • b. O conjunto de instruções usadas por um programa.
  • c. O conjunto de diferentes comandos de uma linguagem de programação de alto nível.
  • d. O conjunto de diferentes instruções de máquina que o processador é capaz de executar.
  • e. O conjunto de diferentes instruções de montagem da linguagem de montagem.
Resposta: d. ✅

Resolução:

  • ISA define todas as instruções que um processador pode executar, incluindo o formato das instruções, os tipos de operações suportadas (como aritméticas, lógicas, controle de fluxo, etc.), os modos de endereçamento, e os registradores disponíveis. Em outras palavras, a ISA é a interface entre o software e o hardware, especificando como o software pode controlar o hardware.

Pergunta 5

Assinale a alternativa INCORRETA:

  • a. A linguagem de montagem é específica de cada processador (ou família de processadores).
  • b. A linguagem de alto nível é independente de plataforma, isto é, não é específica de cada processador (ou família de processadores).
  • c. Um programa em uma linguagem de alto nível precisa ser transformado para instruções de máquina para ser executado pelo processador.
  • d. A linguagem de máquina é específica de cada processador (ou família de processadores).
  • e. Um programa em uma linguagem de alto nível precisa passar pelos processos de compilação ou interpretação para ser executado.
  • f. Um programa em uma linguagem de montagem não precisa ser transformado para instruções de máquina para ser executado pelo processador.
Resposta: f. ✅

Resolução:

  • Um programa em uma linguagem de montagem precisa ser transformado (montado) em instruções de máquina para ser executado pelo processador. O processo de montagem é feito por um montador (assembler), que traduz o código de montagem (assembly) para código de máquina (binário), específico para o processador.

Pergunta 6

Assinale a alternativa que indica o software responsável por combinar um ou mais arquivos contendo instruções de máquina (módulos objeto) em um único programa executável.

  • a. Interpretador.
  • b. Ligador (ou link-editor ou linker).
  • c. Montador.
  • d. Compilador.
  • e. Carregador (loader).
Resposta: b. ✅

Resolução:

  • O ligador (ou link-editor ou linker) é o software responsável por combinar um ou mais arquivos contendo instruções de máquina (módulos objeto) em um único programa executável. Ele resolve referências entre módulos, combina todos os módulos objeto em um único arquivo executável e ajusta os endereços relativos.

Pergunta 7

Assinale a alternativa que NÃO representa uma categoria de instruções de máquina dos processadores.

  • a. Desvios incondicionais.
  • b. Decodificação de instruções.
  • c. Operações lógicas e aritméticas.
  • d. Movimentação de dados entre registradores e memória.
  • e. Desvios condicionais.
Resposta: b. ✅

Resolução:

  • A decodificação de instruções é um processo interno do processador que interpreta as instruções de máquina para determinar quais operações devem ser realizadas.

Pergunta 8

Assinale as alternativas que indicam onde os valores dos operandos, fonte e destino de uma instrução de máquina podem estar localizados.

  • a. Numa função do programa compilador.
  • b. Nos registradores do processador.
  • c. Na área de dados do programa na memória.
  • d. Em um campo imediato da própria instrução de máquina.
  • e. No barramento do computador.
Resposta: b, c, d. ✅

Resolução:

  • Os operandos podem estar nos registradores do processador (b), onde são rapidamente acessíveis.
  • Podem também estar na área de dados do programa na memória (c), embora o acesso seja mais lento que aos registradores.
  • Além disso, operandos podem ser valores imediatos, incluídos diretamente em um campo imediato da própria instrução de máquina (d), permitindo acesso rápido sem consultar a memória.

Pergunta 9

As instruções de máquina representam as operações que são interpretadas e executadas pelo processador.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Verdadeiro. ✅

Resolução:

  • As instruções de máquina são conjuntos de comandos codificados em binário que o processador interpreta e executa diretamente. Cada instrução de máquina especifica uma operação básica, como aritmética, lógica, movimentação de dados, ou controle de fluxo, que o processador realiza.

Pergunta 10

Assinale a alternativa que indica o software responsável por traduzir um programa em uma linguagem de programação de alto nível para instruções de máquina.

  • a. Compilador.
  • b. Carregador (loader).
  • c. Ligador (ou link-editor ou linker).
  • d. Montador.
Resposta: a. ✅

Resolução:

  • Um compilador é um software responsável por traduzir programas escritos em linguagens de alto nível, como C, C++, Python, Java, entre outras, para linguagem de máquina, que é entendida e executada diretamente pelo processador.
