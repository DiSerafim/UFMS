Expressões Numéricas

EX:
x = 2 + 6 + 2 + 2·4³ - √16
x = 2·{23 - 2[48 + 2⁽⁵⁻²·²⁾]} - 8

Em uma expressão numérica sem os simbolos gráficos, como é o caso da primeira expressão, devemos seguir a seguinte ordem:
1º Resolver as potências e raizes.
2º Resolver as multiplicações e divisões.
3º Resolver as somas e subtrações.

Em uma expressão numérica com os simbolos gráficos, como é o caso da segunda expressão, devemos seguir a seguinte ordem:
1º Resolver as operações dentro de parênteses.
2º Resolver as operações dentro de colchetes.
3º Resolver as operações dentro das chaves.

Ordem das operações

1ª expressão

x = 2 + 6 ÷ 2 + 2·4³ - √16
x = 2 + 6 ÷ 2 + 2 · 64 - 4
x = 2 + 3 + 128 - 4
x = 129„

2ª expressão

x = 2·{23 - 2[48 + 2⁽⁵⁻²·²⁾]} - 8
x = 2·{23 - 2[48 + 2¹]} - 8
x = 2·{23 - 2·50} - 8
x = 2·{23 - 100} - 8
x = 2 · -77 - 8
x = -154 -8
x = -162„

Exemplos:

1) Pedro tinha 7 caixas de CDs, com 16 CDs em cada uma, após vender um oitavo dos CDs, de duas caixas, ele ficou com quantos CDs?
Vamos denotar por x a quantidade de CDs.
x = 7·16 - 1/8 · (2·16)
x = 7·16 - 1/8 · 32
x = 112 - 4
x = 108„

2) Determine o valor numérico do polinômio:
p(x) = 3x⁴ + 4x³ - 2x² - 3, para x = -2
p(-2) = 3(-2)⁴ + 4(-2)³ - 2(-2)² - 3
p(-2) = 3·16 + 4(-8) - 2·4 - 3
p(-2) = 48 - 32 - 8 - 3
p(-2) = 5„

3) Para x = 4, qual é o valor da função f(x) = √x²+9-3 / 2x + 4
f(4) = √4²+9-3 / 2·4 + 4
f(4) = √25-3 / 8+4
f(4) = 5-3 / 12
f(4) = 2/12 (simpĺificamos dividindo por 2)
f(4) = 1/6„

4) Na produção de um determinado produto, uma empresa utiliza 2 placas de circuitos, 6 metros de fio, 25 pontos de solda, 16 parafusos e 1 console. Custos dos componentes:
Placa de circuito - R$ 25,00 cada uma.
Fio - R$ 120,00 o rolo com 100.
Solda - R$ 3,00 cada ponto.
Parafuso - R$ 10,00 o cento.
Console - R$ 15,00.
Qual o custo na fabricação de 200 produtos?

x = 200·(2·25 + 6·120/100 + 25·3 + 16·10/100 + 15)
x = 200·(50 + 6·1,2 + 25·3 + 16·0,1 + 15)
x = 200·(50 + 7,2 + 75 + 1,6 + 15)
x = 200·148,8
x = 29760

O custo é de R$ 29.760„

Importância da ordenação

Resolva as seguintes Expressões numéricas.

a) x = 3·5 + 5 ÷ 5 - 2
x = 15 + 1 - 2
x = 14„

b) x = 3·(5 + 5 ÷ 5 - 2)
x = 3·(5 + 1 - 2)
x = 3·4
x = 12„

c) x = (3·5 + 5) ÷ (5 - 2)
x = (15 + 5) ÷ 3
x = 20/3

d) x = 3·[5 + (5 ÷ 5)] - 2
x = 3·[5 + 1] - 2
x = 3·6 - 2
x = 18 - 2
x = 16„

e) x = 3·{[(5 + 5) ÷ 5] - 2}
x = 3·{[10 ÷ 5] -2}
x = 3·{2 - 2}
x = 3·0
x = 0„
