Propriedades da adição

a + b = b + a => Comutativa

(a + b) + c = a + (b + c) => Associativa

a + 0 = a => Elemento neutro

Se a > b, então a + c > b + c

Se a + b = a então b = 0

Multiplicação

Exemplo

3 · 2 = 2 + 2 + 2 = 6

23 · 47 = 1081

a · b = b · a => comutativa

(a · b) · c = a · (b · c) => associativa

a · 1 = a => elemento neutro

Se a > b, então a · c ≥ b · c

Se a · b = a, então b = 1 ou a = 0

Propriedade Distribuitiva

Para quaisquer números naturais, a, b, c, vale:

(a + b) · c = a · c + b · c

Potência de número Natural

Dados dois números naturais a e b definimos a potência de base a e expoente b, como sendo o número natural:

aᵇ = a·a... a, se b ≠ 0
└─> se a ≠ 0 e b = 0, definimos a = 1
Obs: No caso a = b = 0 é indeterminado.
