Divisão de Frações

Algotitimo:
a/b : c/d = a:c/b:d
a/b : c/d = a/b · d/c

b ≠ 0
c ≠ 0
d ≠ 0

Exemplo:

a) 3/5 : 3 = 3/5 : 3/1 = 1/5

b) 2/3 : 2 = 2/3 : 2/1 = 1/3

c) 6/7 : 6/5 = 6/7 · 5/6 = 1/7 · 5/1 = 5/7, Nesse caso, não é possível dividir 7/5 então repetimos a primeira e multiplica pelo inverso do segundo, 6/6 resulta em 1

d) 35/12 : 7/4 = 5/3
