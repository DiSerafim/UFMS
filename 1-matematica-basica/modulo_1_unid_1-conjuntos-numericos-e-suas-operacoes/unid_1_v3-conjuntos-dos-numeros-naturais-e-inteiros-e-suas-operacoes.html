Números Inteiros

Determine a solução da equação x + 3 = 0, para:

x ∈ N
Não existe solução, pois para todo x natural, x + 3 ≥ x.

x assumindo valor negativo
A solução é x = -3, pois -3 + 3 = 0.

Conjunto dos números inteiros

Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}

Exemplo

Dado a ∈ Z, existe um b ∈ Z, tal que a + b = 0
O número b é chamado o oposto de a e é denotado por -a.

-2 é o oposto de 2, pois -2 + 2 = 0.

Valor absoluto ou (módulo)

Dado um número inteiro a, definimos por seu valor absoluto ou (módulo) ao número natural |a| definido do seguinte modo:

|a| = { a; se a ≥ 0
└-> { -a; se a < 0

Exemplo

|5| = 5
|-3| = 3

Adição nos inteiros

Dados dois números inteiros a e b definimos sua soma da seguinte forma:

Se a e b têm o mesmo sinal, efetuamos a soma dos valores absolutos de a e b e mantemos o mesmo sinal

Se a e b possuem sinais contrários, calculamos a diferença entre seus valores absolutos e mantemos o sinal daquele que possui o maior valor absoluto

Exemplo

3 + 8 = 11
-4 + (-2) = -6
-5 + 12 = 7
7 + (-10) = -3

Multiplicação nos Inteiros

Dados dois números inteiros a e b definimos seu produto da seguinte forma:

- Se a e b têm o mesmo sinal,
a·b = |a||b|
- Se a e b possuem sinais contrários,
a·b = -|a||b|

Exemplo

3·7 = 21
(-2)·(-8) = 16
(-4)·3 = -12

Equação Diofantina

Determine números inteiros a e b, tais que 8a + 5b = 1

Se a = 1 a equação 8 + 5b = 1 => 5b = -7, não tem solução inteira, pois -7 não é multiplo de 5.

Se a = 2 a equação 16 + 5b = 1 => 5b = -15, tem como solução b = -3.

Assim temos nossa primeira solução que é:
a = 2 e b = -3
a = 2 - 5k k ∈ Z
b = -3 + 8k k ∈ Z

Para k = 1 => a = -3 e b = 5 Para k = 2 => a = -8 e b = 13 Para k = -1 => a = 7 e b = -11
