Números Reais

Um quadrado tem lados medindo 1m, qual é a medida de sua diagonal

 ______1________
 |            /|
1|         /   |
 |      / d    |
 |____/________|
d² = 1²+1² = 2

Devemos encontrar um número d, tal que

d² = 2
1² = 1 e 2² = 4
Logo d não é um número inteiro.
Vamos supor que d ∈ Q, então existem inteiros p e q ≠ 0, com MDC(p; q) = 1, tais que:
d = p/q

d² = 2 =>
(p/q)² = 2 =>
p/q · p/q = 2 =>
p²/q² = 2 =>
p² = 2q²

Logo p2 é um número par, como p é um número inteiro, p também é par.

Dessa forma, p = 2k para algum k ∈ Z e assim, substituindo na equação anterior, temos:
p² = 2q² =>
(2k)² = 2q² =>
2q² = 4k² => simplificamos por 2
q² = 2k² =>

Logo q também é par e assim MDC(p; q) ≥ 2, o que é um absurdo pois supomos p e q primos entre si.
Assim, se x é tal que x² = 2, temos que x ∉ Q.
Ou seja, x é um número irracional.

Números Irracionais

O número π = 3,14159265358979323846...
O número e = 2,71828182845904523536028...
O número φ = 1,6180339...

1) Números naturais: são representados pelo conjunto "N" dentro do diagrama de Venn. O conjunto N é uma região delimitada dentro de um retângulo e contém apenas números inteiros positivos. 

    1  2  3  4  5  ...     
    +--------------------+ 
N  | ●  ●  ●  ●  ●  ... |  
    +--------------------+

2) Números racionais: são representados pelo conjunto "Q" dentro do diagrama de Venn. O conjunto Q é uma região que está contida dentro do conjunto de números reais "R" e inclui todas as frações que podem ser escritas na forma "p/q", onde "p" e "q" são inteiros e "q" é diferente de zero. 

        R                  
+------------------------+ 
|         Q              | 
+------------------------+

3) Números irracionais: são representados pelo conjunto "I" dentro do diagrama de Venn. O conjunto I é uma região que está contida dentro do conjunto de números reais "R" e inclui todos os números reais que não são racionais. 

        R                  
+------------------------+ 
|            I           | 
+------------------------+

4) Números reais: são representados pelo conjunto "R" dentro do diagrama de Venn. O conjunto R é a região que inclui todos os números reais, incluindo números racionais e irracionais. 

 +------------------------+ 
 |            R           | 
 +------------------------+

Propriedades nos Reais

Adição: Sejam a, b e c números reais.

1) a+b = b+a comutativa

2) (a+b) + c = a + (b+c) associativa

3) a+0 = a elemento neutro

4) Dado a ∈ R, existe um único y ∈ R tal que a+y = 0, y é chamado elemento oposto de a e é denotado por -a.

Multiplicação: Sejam a, b e c números reais.

1) a·b = b·a comutativa

2) (a·b) · c = a · (b·c) associativa

3) a·1 = a elemento neutro

4) Dado a ≠ 0, existe um único y ∈ R tal que a·y = 1, x é chamado inverso de a e será denotado por 1/a ou a-¹.

<3>Demais Propriedades

- Para quaisquer números reais, a, b, c, temos (a+b)·c = a·c + b·c, propriedade distributiva.
- Para qualquer a real, -a = (-1)·a e -(-a) = a
- Para todo a > 0, aᴾ·aᵟ = aᴾ⁺ᵟ e (aᴾ)ᵟ = aᴾ·ᵟ
- Para todo a > 0, b > 0, (a·b)ᴾ = aᴾ·bᴾ
- Para todo ᴬ⁄ᴮ > 0, (ᴬ⁄ᴮ)ᴾ = aᴾ/bᴾ

Exemplo:

1. (∛8/27)⁻² = (∛8/∛27)⁻² = (2/3)⁻² = (3/2)² = 9/4

2. x²(x³ + (x⁴)¹⁄²) = x² (x³ + x⁴⁄²) = x⁵ + x⁴

3. -3·(2x - x²) = -6x + 3x²
