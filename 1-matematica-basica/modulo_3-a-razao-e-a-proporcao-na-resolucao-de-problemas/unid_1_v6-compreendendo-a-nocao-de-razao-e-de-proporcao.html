Proporções

Problema 1:
Em uma salina, de cada 1m³ de água salgada, são retirados 40dm³ de sal.
Considerando mantida esta razão, para obter 2m³ de sal, quantos m³ de água salgada é necessária?

Sal = s
Água salgada = a
x = 40dm³/1m³ dividimos por mil para calcular o decimetro e tranformar em metro
[40/1000]/1m³
0,04m³/1m³
0,04/1 = 2/x passamos o x pra lá e mutiplicamos
x·0.04 = 2·1
x = 2/0.04
x = 2 / 4/100 passamos o 100 que está dividindo para lá, multiplicando
x = 2·100/4
x = 200/4
x = 50m³„
