Problema 1:
A razão entre as medidas do comprimento e da largura de uma sala retangular é 5/7.
Determine as dimensões da sala, sabendo que a soma delas é 84m.

l = 5
c = 7
d = 5/7
d = c + l = 84m
d = c/l + 1 = 5/7 + 1 adicionamos o 1 mas poderia ser qualquer outro ex:
d = c/l + l/l = 5/7 + 7/7 aqui trocamos o 1 pelo al/l e 7/7
d = c+l/l => 12/7
d = 89/l => 12/7 então qual o valor que multiplicado por 12 é igual a 89?
d = 89/l => [(7 · 12) / (7 · 7)]
d = 89/49„

Propriedade 1:

Em uma proporção, a soma dos dois primeiros termos está para o segundo termo assim como a soma dos dois últimos termos está para o quarto termo.

a/b = c/d
a/b + 1 = c/d + 1 inverteremos o 1 para qualquer outro
a/b + b/b = c/d + d/d
a+b/b = c+d/d
