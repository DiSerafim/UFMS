Proporções

Significa uma relação entre as partes de uma grandeza.
Proporção é uma igualdade de razões.

Elementos de uma proporção

Dados 4 números a, b, c e d ∈ ℚ, nesta ordem, dizemos que eles formam uma proporção.
Quando a razão entre o 1º e o 2º é igual a razão entre o 3º e o 4º, ou seja.
a/b = c/d
ou
a:b = c:d
A leitura é assim:
a está para b assim como c está para d.

Em uma proporção, o produto dos extremos é igual ao produto dos meios.

a/b = c/d <=> a·d = b·c
