Pergunta 1

Na videoaula, Fundamentos de Internet e Web - Protocolos de comunicação da internet do professor Júlio Cezar Estrella, são apresentados os protocolos fundamentais de comunicação da Internet, quais das alternativas NÃO fazem parte dos apresentados?

Escolha uma ou mais:

  • a. MMS
  • b. UDP
  • c. IP
  • d. H.264
  • e. TCP
Resposta: a. MMS & d. H.264. ✅

Resolução:

  • MMS é um serviço de envio de mensagens multimídia e H.264 é utilizado para compressão de vídeo, não como um protocolo de rede.

Pergunta 2

De acordo com a definição de (MARINHO, 2019) qual é o protocolo utilizado para a transferência de documentos hipermídia (hipertexto) na Internet?

  • a. SSH
  • b. TCP/IP
  • c. HTTP
  • d. FTP
  • e. SFTP
Resposta: c. HTTP. ✅

Resolução:

O HTTP é o protocolo responsável pela comunicação entre navegadores e servidores para a transferência de páginas web.

Pergunta 3

Na videoaula, Fundamentos de Internet e Web - Introdução às Redes de Computadores do professor Júlio Cezar Estrella é afirmado que a internet não possui protocolos de comunicação, essa afirmação é verdadeira ou falsa?

Escolha uma opção:

Verdadeiro ou Falso.

Resposta: Falso. ✅

Resolução:

  • A Internet possui diversos protocolos de comunicação, como o TCP (Transmission Control Protocol), IP (Internet Protocol), UDP (User Datagram Protocol), entre outros, que são essenciais para a transferência de dados e a comunicação entre dispositivos na rede.

Pergunta 5

Levando em consideração os novos recursos do HTML5 apontados por Teruel (2014), selecione a alternativa que NÃO representa um novo recurso do HTML5.

  • a. Os elementos <audio> e <video> para reprodução de mídia.
  • b. Novos atributos para campos de formulário e tipos de dados, como date, time, email, url, search etc.
  • c. Novos elementos de conteúdo específico, como <article>, <footer>, <header>, <nav>, <section>.
  • d. Compatibilidade com XHTML.
  • e. O elemento <canvas> para criação de desenhos e gráficos.
Resposta: d. Compatibilidade com XHTML. ✅

Resolução:

  • HTML5 trouxe novos elementos e atributos para melhorar a semântica e a funcionalidade das páginas, mas a compatibilidade com XHTML não é considerada um novo recurso do HTML5, pois XHTML é uma versão mais estrita do HTML baseada em XML.

Pergunta 6

De acordo com a definição de (MARINHO, 2019) sobre Client/Server Side, o Firefox é um elemento do Server Side.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Falso. ✅

Resolução:

  • O Firefox é um navegador web, que é um exemplo de um elemento do Client Side (lado do cliente). Ele executa scripts e processa páginas web no lado do cliente, permitindo a interação com o usuário e a visualização de conteúdos web. Elementos do Server Side (lado do servidor) são aqueles que processam dados e gerenciam o backend de aplicações web, como servidores web e bancos de dados.

Pergunta 8

Na videoaula, Fundamentos de Internet e Web - Introdução às Redes de Computadores do professor Júlio Cezar Estrella é afirmado que: apesar de baseados no modelo OSI de referência o modelo de fato no qual a internet se baseia é o TCP/IP, essa afirmação é verdadeira ou falsa?

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Verdadeiro. ✅

Resolução:

  • Embora o modelo OSI (Open Systems Interconnection) seja um modelo teórico de referência que descreve a comunicação em redes de computadores, a Internet de fato se baseia no modelo TCP/IP (Transmission Control Protocol/Internet Protocol) para a comunicação de dados. O modelo TCP/IP é mais prático e amplamente utilizado na implementação real da Internet.

Pergunta 9

Na videoaula, Fundamentos de Internet e Web - Linguagens de marcação para a WWW do professor Júlio Cezar Estrella é afirmado que uma linguagem de marcação é responsável por permitir que o navegador web apresente corretamente uma página, essa afirmação é?

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Verdadeiro. ✅

Resolução:

  • Uma linguagem de marcação, como HTML (HyperText Markup Language), é responsável por estruturar e formatar o conteúdo de uma página web, permitindo que o navegador web a apresente corretamente. As tags e atributos dessa linguagem indicam como o conteúdo deve ser exibido, como cabeçalhos, parágrafos, links, imagens e outros elementos.

Pergunta 10

De acordo com a definição de (MARINHO, 2019) sobre Client/Server Side, o Microsoft Internet Information Service é um elemento do Client Side.

Escolha uma opção: Verdadeiro ou Falso.

Resposta: Falso. ✅

Resolução:

  • O Microsoft Internet Information Services (IIS) é um servidor web, que é um exemplo de um elemento do Server Side (lado do servidor). O IIS é responsável por hospedar e gerenciar sites e aplicações web, processando as solicitações enviadas pelos clientes (navegadores) e retornando as respostas apropriadas. O Client Side (lado do cliente) refere-se aos componentes que executam no navegador web do usuário, como HTML, CSS e JavaScript.
